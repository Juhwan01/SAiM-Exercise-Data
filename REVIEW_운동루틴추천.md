# 운동 루틴 추천 룰베이스 — 한 줄 한 줄 회의용 정리

> 진입 함수: `rule_engine.recommend_routine(profile) -> RoutineRecommendation`
> 룰 수: 97개 (`routine_rules/` 12개 모듈)
> 이 문서는 각 룰이 **어떤 입력을 보고 / 어떤 분기로 / 어떤 출력을 내는지**만
> 사실 그대로 옮긴 회의 자료다. 옳고 그름 판단은 회의에서.

---

## 0. 전체 흐름 (`recommend_routine.py`)

입력으로 `UserProfile` 1개를 받고, 13단계를 순서대로 실행한다.
1단계에서 차단 신호가 나오면 거기서 끝, 처방 정지 + 의료 의뢰만 남긴다.

```
[1] 안전 게이트 (L02-L08)
[2] 레벨 분류 (L04)
[3] 영양 (K19/K20 → K22 → K21)
[4] 분할/프로그램 (D05 → D06/D07/D08/D10)
[5] 시작 무게 (D11/D12 + D14 클램프 + D15 증량)
[6] 강도/볼륨 (C02/C05/C06/C07/B02/B05)
[7] 주기화/디로드 (D02/D03 + G01/G05/G10 + I01)
[8] 운동 선택 (J05/J03/J02/I03 + J01)
[9] 워밍업/스트레칭 (I05/I06/P01/P03/P04/P05)
[10] 카디오 (O01/O02/O03)
[11] 회복/수면 (K05/K06/K07)
[12] 부상 이력 (Q04/Q05/Q06)
[13] 행동·교육 (R10/N02/N04/M03)
```

## 1단계 — 안전 게이트 (`routine_rules/populations.py`)

### L02 — 노인 (50+)
- 출처: ACSM 2026 / ESPEN ★★★
- **조건**: `profile.나이 ≥ 50` 또는 `질환⊃"고령(50+)"`
- **결과 (처방 dict)**:
  - 주 빈도 2~3 / 렙 8~12 / 세트 2~3 / %1RM 60~80
  - 단백질 1.2~1.6 g/kg
  - 메모: 첫 8주 적응기 점진과부하 보수적
- 추가 경고: "점진과부하 보수적 진행 — 8주 적응기 필요"

### L03 — 청소년 (<18)
- 출처: NSCA Youth Position Statement ★★★
- **조건**: `profile.나이 < 18` 또는 `질환⊃"청소년(<18)"`
- **결과**: 1RM 테스트 잠금, 강도는 RPE 기반, 렙 6~15
- 주의 리스트: 성장판 안전 / 폼 우선 / 코치 감독 필수

### L05 — 제2형 당뇨 처방
- 출처: ACSM 2022 + ADA 2024 + 대한당뇨병학회 2023 ★★★
- **조건**: `질환⊃"당뇨"`
- **결과 (처방 dict)**:
  - 유산소 빈도 3~7회/주, 연속 비활동 ≤ 2일
  - 저항 빈도 2~3, RPE 상한 7
  - 유산소 75~150분/주
  - 저항 종목 8~10개, 세트 1~3
  - 권장: 걷기·자전거·수영·기계식 저항
  - 식후 30분 가벼운 걷기 권고
  - 회피 조건: 증식성 망막증·자율신경병증·공복혈당>250+케톤

### L06 — 고혈압 처방
- 출처: ACSM 2022 + 대한고혈압학회 2022 ★★★
- **조건**: `질환⊃"고혈압"` + 적색기 BP 검사
- **결과 (처방 dict)**:
  - 유산소 5~7회, 저항 2~3회
  - 저항 RPE 상한 6, %1RM 60~70, 렙 8~12, 세트 2~4
  - 유산소 30분/일 (90~150분/주)
  - Valsalva 금지 = True, 안정시 BP 절대상한 (180/110)
- **추가 차단**: `bp_sbp ≥ 180` 또는 `bp_dbp ≥ 110` → 안전_차단=True, 의료 의뢰

### L07 — 이상지질혈증 처방
- 출처: ACSM 2022 + 한국지질학회 2022 ★★★
- **조건**: `질환⊃"이상지질혈증"`
- **결과**:
  - 유산소 5~7회, 저항 2~3회, 유연성 2~3회
  - RPE 상한 6, 유산소 200~300분/주(최소 150)
  - 주당 1,000~2,000 kcal 소비
- **추가**: `ldl > 190` → 의료 의뢰 추가

### L08 — 골관절염 처방
- 출처: OARSI 2019 + ACR 2019 ★★★
- **조건**: `질환⊃"골관절염_무릎"` 또는 `"골관절염_고관절"`
- **결과**:
  - 저항 2~3, 유산소 3~5, 유연성 매일
  - 저항 RPE 5~7, 통증 NRS 상한 4
  - %1RM 60~70, 렙 8~15
  - 무릎 ROM 제한: 굴곡 ≤ 90°
  - 회피: 깊은 스쿼트·점프·착지·중량 런지
  - 권장 옵션: 수중운동·태극권

## 3단계 — 영양 (`routine_rules/nutrition.py`)

### K19 — Mifflin-St Jeor BMR
- 출처: Mifflin & St Jeor 1990 ★★★
- **조건**: `profile.체지방률_pct` **없음**
- **공식**:
  - 남: `10×W + 6.25×H − 5×나이 + 5`
  - 여: 동일식 − 161
- **결과**: BMR 정수 kcal

### K20 — Katch-McArdle BMR
- 출처: Aragon ISSN 2017 ★★★
- **조건**: `profile.체지방률_pct` **있음**
- **공식**: `370 + 21.6 × LBM`, LBM = 체중 × (1 − 체지방률/100)

### K22 — 활동 카테고리 매핑
- 출처: FAO/WHO/UNU 2001 ★★☆
- **조건**: 항상 발동
- **분기**:
  - 육체노동 + (주3~5회 또는 주6~7회) → "매우_활발"
  - 주6~7회 → "활발"
  - 주3~5회 → "보통"
  - 주1~3회 → "가벼움"
  - 그 외 → "좌식"

### K21 — 활동계수(PAL) → TDEE
- 출처: FAO/WHO/UNU 2001 ★★★
- **공식**: `TDEE = BMR × PAL`
- **PAL 표**: 좌식 1.2 / 가벼움 1.375 / 보통 1.55 / 활발 1.725 / 매우 활발 1.9

### 목표 칼로리 가산 (룰 외 진입 함수 로직)
- 다이어트 −500 / 벌크업 +300 / 근육량 증가 +200 / 그 외 0

## 4단계 — 분할/프로그램 (`routine_rules/periodization.py`)

### D05 — 분할 선택
- 출처: Schoenfeld 2016 / Israetel ★★★
- **분기 (가용일 + 레벨)**:
  - 가용일 ≤ 3 또는 레벨="초보" → "풀바디_주N"
  - 가용일 = 4 → "상하분할_주4"
  - 가용일 ≥ 5 → "PPL_주N"

### D06 — Starting Strength (초보 기본값)
- 출처: Rippetoe ★★★
- **결과**: A루틴(스쿼트 5x3 / 벤치 5x3 / 데드 1x5), B루틴(스쿼트 / OHP / 파워클린 또는 바벨로우)
- 세션당 +2.5~5kg, 6~12개월 운영

### D07 — 5/3/1 Wendler
- 출처: Wendler ★★★
- **조건**: 목표="스트렝스" + 레벨="중급"
- **공식**: TM = 1RM × 0.9
- **주차 표**:
  - 1주: 5x3 (65/75/85% TM)
  - 2주: 3x3 (70/80/90)
  - 3주: 5/3/1+ AMRAP (75/85/95)
  - 4주: 디로드 5x3 (40/50/60)

### D08 — GZCLP
- 출처: r/Fitness wiki ★★☆
- **조건**: 목표="스트렝스", 위 D07 조건 미달 시
- **결과**: T1 (메인 5+렙) / T2 (보조 10+렙) / T3 (아이솔 15+렙) / 주 4회

### D10 — PHUL/PHAT
- 출처: Norton / Campbell ★★☆
- **조건**: 목표="벌크업/근육량 증가" + 레벨="중급" + 가용일 ≥ 4
- **분기**: 가용일 ≤ 4 → PHUL / 5+ → PHAT

### 진입 함수의 폴백
- 위 조건 모두 못 만족 → D06 (Starting Strength)

> **회의 포인트**: D01 (Linear Periodization), D03 (Block Periodization), D09 (nSuns), D04 (Flexible DUP) 는 모듈에 함수 정의는 있지만, 진입 함수의 분기에서 직접 선택되지 않는다. 다른 분기(주기화 단계)에서 일부만 호출됨.

## 5단계 — 시작 무게 (`periodization.py`)

### 분기 1: 사용자 레벨 ≠ "초보"
- `profile.one_rm_kg` 있으면 → 시작무게 = 1RM × 0.8 (각 종목)
- 없으면 → 단계 스킵

### D11 — 남성 초보 시작 무게
- 출처: StrongLifts 5x5 ★★☆
- **조건**: 레벨=초보 + 성별=남
- **결과**: {스쿼트 20, 벤치 20, OHP 20, 데드 40} (kg, 빈 바벨 기준)

### D12 — 여성 초보 시작 무게
- 출처: IPF Technical Rules 2024 ★★★
- **조건**: 레벨=초보 + 성별=여
- **분기**: `headers헬스장_여성바_15kg` 여부
  - 있음 → bar=15kg, 데드 35kg
  - 없음 → bar=20kg, 데드 35kg

### D14 — 첫 세션 워크셋 클램프
- 출처: Rippetoe ★★☆
- **공식**: 종목별 캡(스쿼트 65/35, 데드 70/40, 벤치 40/25, OHP 30/18) × (체중 / 기준체중)
  - 기준체중: 남=70, 여=55
- **분기**: 요청 무게 > 캡 → 캡으로 떨굼 + 메모

### D15 — 증량 폭 (초기)
- 출처: Rippetoe ★★☆
- **결과 (kg/세션)**:
  - 하체(스쿼트·데드): 초기 +10, 이후 +5
  - 상체(벤치·OHP·로우): 초기 +5, 이후 +2.5
- 진입 함수는 `is_early_phase=True` 로 초기 폭만 출력

## 6단계 — 강도/볼륨 (`intensity.py` + `volume.py`)

### C02 — Repetition Continuum
- 출처: Schoenfeld 2021 / NSCA ★★★
- **분기 (목표)**:
  - 스트렝스 → 렙 1~6, %1RM 85~100
  - 벌크업/근육량 증가 → 렙 6~12, %1RM 67~85
  - 다이어트 → 렙 8~15, %1RM 60~80
  - 그 외(건강증진) → 렙 12~20, %1RM 50~67

### C05 — 디폴트 RIR
- 출처: Refalo / Pelland 2024 ★★★
- **분기**:
  - compound → 2
  - isolation → 1
- 진입 함수는 항상 "compound" 로 호출 → RIR=2

### C06 — 최소 빈도
- 출처: Schoenfeld 2016/2018 ★★★
- **결과**: 근육군당 주 최소 2회 (고정값)

### C07 — 휴식 시간 (루틴 컨텍스트)
- 출처: Schoenfeld 2016 ★★★
- **분기 (목표)**:
  - 스트렝스 → {compound: 240, isolation: 180}
  - 그 외 → {compound: 150, isolation: 90}

### B02 — MEV 시작 볼륨
- 출처: Israetel ★★★
- **결과 (근육군당 세트/주)**: 가슴 8 / 등 10 / 어깨 8 / 이두 8 / 삼두 6 / 다리 7 / 둔근·햄 6 / 복근 0 / 기타 8
- 진입 함수는 6대 근육군(가슴·등·어깨·다리·이두·삼두)에 일괄 적용

### B05 — Volume Dose-Response
- 출처: Schoenfeld / Krieger ★★★
- **분기 (목표 → 주당 권장 볼륨 범위)**:
  - 벌크업/근육량 증가 → 12~20
  - 스트렝스 → 8~16
  - 다이어트 → 10~18
  - 건강증진 → 8~12

> **회의 포인트**: C01·C03·C04·C08·C09·C10·C11·C12·C13·C14·C15, B01·B03·B04 는 모듈에 함수가 있으나 진입 함수에서 호출되지 않음. 시작 무게/세션 가이드/실패 처리에 추후 진입 가능.

## 7단계 — 주기화/디로드 (`periodization.py` + `recovery.py` + `knowledge.py`)

### 레벨 분기
- 초보 → I01 (Linear Progression 사용)
- 중급 → D02 (DUP)
- 고급 → D03 (Block Periodization)

### D02 — DUP (중급)
- **결과**: [월=비대(8~12렙, 67~75%), 수=근력(3~6렙, 85~92%), 금=파워(1~3렙, 90~100%)]

### D03 — Block (고급)
- **결과**: 축적(1~4주, 볼륨↑) → 변환(5~7주, 강도↑) → 실현(8~10주, 최대 강도)

### G01 — 디로드 정의
- **결과**: 디로드 5~7일, 주기 4~6주

### G10 — 한국 사용자 디로드 주기
- **분기**:
  - 초보/절대초보 → 신호기반 자동 (기본 주기 None)
  - 그 외 → 강제, 기본 6주
- **추가**: 진입은 "5주 + 1주 디로드 = 메조 6주" 로 박음

### G05 — 디로드 정량 처방
- **결과**: 세트 ½, 무게 ×0.95, 7일

> **회의 포인트**: D01·D04·G03·G04·G06·G08·G09·G11 는 함수 정의만 있고 호출 안 됨.

## 8단계 — 운동 선택 (`exercise_select.py`)

### J05 — 컴파운드:아이솔 비율
- **분기 (레벨)**: 초보 70:30 / 중급 60:40 / 고급 50:50

### J03 — 자유중량:머신 비율
- **조건**: `prefer_safety = (당뇨 또는 고혈압)`
- **분기**:
  - 초보 또는 prefer_safety → 40:60
  - 스트렝스 → 80:20
  - 그 외 → 60:40

### J02 — 장비 매칭
- **조건**: 운동별로 사용자 장비 리스트(`profile.장비`) 검사
- **분기**: 바벨 있음 → 원래 운동 / 없으면 덤벨→케틀벨→밴드→맨몸 순서로 대체

### I03 — 운동 순서 (컴파운드 우선)
- **결과**: 가중치 표(빅3 컴파운드=1, 보조=2, 머신=4, 아이솔=5)로 정렬

### J01 — 부상별 대체
- **조건**: `profile.부상이력` 의 부위별 매칭
- **매트릭스**:
  - 무릎 → 백스쿼트 → 레그프레스/벨트스쿼트/스플릿스쿼트
  - 어깨 → 벤치 → 푸시업/덤벨프레스/랜드마인
  - 허리 → 데드 → 힙쓰러스트/RDL/백 익스텐션
  - 손목 → 백스쿼트 → safety bar/프론트스쿼트
  - 팔꿈치 → 벤치 → 체스트프레스 머신/덤벨플라이

> **회의 포인트**: F05 (SFR), J04 (편측/양측), J07 (근육군 매핑), E14, E17 함수는 정의만 있고 진입 호출 X.

## 9단계 — 워밍업/스트레칭 (`warmup_stretch.py`)

### I05 — 일반 워밍업
- **공식**: `min(10, max(5, 세션가용분 // 6))` 분
- **결과**: "N분 저강도 유산소 — 땀 살짝 날 정도"

### I06 — 컴파운드 워크업
- **조건**: 시작 무게에 "스쿼트" 있음 (스쿼트 무게 기준)
- **결과**:
  - 빈 봉 5렙 × 2세트
  - 40% × 5렙 / 60% × 3렙 / 80% × 2렙
  - 본 세트

### P01 — 워밍업 스트레칭 정책
- **결과**: 정적 회피, 동적 권장, 정적은 운동 후

### P05 — 정적 스트레칭 경고 (메시지)
- **결과**: "워밍업 정적 스트레칭 ≥45초 회피 — 최대 근력 평균 -5.4%"

### P03 — SMR

## 12단계 — 부상 이력별 (`injury_recovery.py`)

> **공통 호출 인자**: `days_since_onset=30, pain_nprs=2` (진입에서 하드코딩)

### Q04 — 요추 통증
- 출처: NICE NG59 + Foster 2018 ★★★
- **분기 (경과일)**:
  - 적색기 신호(다리 양측 저림·안장마비·괄약근 이상·배뇨이상·발열·암 병력·외상) 1개라도 → 차단, 응급실/신경외과
  - < 3일 → 일상 활동 + 골반 틸트 + McKenzie 신전 / 데드·스쿼트 회피
  - 3~14일 → 스트레칭·근력·유산소·운동조절
  - ≥ 14일 → 글루트·코어 활성화 + 1RM 30%부터, 주 +10% 증가
- 진입에서 30일 입력 → "복귀 14일~" 단계 적용

### Q05 — 어깨 RCRSP
- 출처: Lewis 2016 / Cools 2015 ★★★
- **분기**:
  - 적색기 신호 → 차단, 정형외과
  - < 7일 → 오버헤드 회피, 진자운동
  - 7~42일 → 밴드 외회전·서라투스 펀치·로우, NPRS 상한 4
  - ≥ 42일 → 벤치/OHP/풀업 1RM 30%부터

### Q06 — 무릎 PFPS
- 출처: Powers 2017 / Crossley 2016 ★★★
- **분기**:
  - 적색기 → 차단, 정형외과
  - < 14일 → 딥 스쿼트·런지·오픈체인 레그익스텐션 회피
  - 14~42일 → 클램쉘·힙 어브덕션·미니 스쿼트 0~45°
  - ≥ 42일 → 스쿼트/런지 점진

> Q02 (단계 전환), Q03 (복귀 기준 4가지 충족) 은 모듈 정의만, 진입에서 호출 X.
