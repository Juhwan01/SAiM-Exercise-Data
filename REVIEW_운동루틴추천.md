# 운동 루틴 추천 룰 — 한 줄씩 같이 읽는 회의 문서

## 이 함수는 언제 호출되나
사용자가 온보딩에서 프로필을 완성했을 때 1회. 받아오는 건 사용자 한 사람 정보(성별·체중·키·나이·훈련 개월·목표·주가용일·세션 가용분·체지방률·1RM·부상이력·질환·혈압·LDL·장비). 돌려주는 건 주간 프로그램 한 묶음(BMR·TDEE·목표 칼로리·분할 형태·추천 프로그램·주간 스케줄·시작 무게·근육군별 주간 세트·디로드 주기·운동 풀·워밍업·카디오·회복 가이드·교육 메시지).

## 흐름
13단계를 순서대로. 1단계에서 차단 나오면 처방 정지하고 의료 의뢰만 띄움.

```
1.  안전 게이트 (50세↑·청소년·당뇨·고혈압·이상지질혈증·골관절염)
2.  레벨 분류 (초보·중급·고급)
3.  영양 (BMR → TDEE → 목표 칼로리)
4.  분할/프로그램 선택
5.  시작 무게 + 증량 폭
6.  강도/볼륨
7.  주기화/디로드
8.  운동 선택
9.  워밍업/스트레칭
10. 카디오
11. 회복/수면
12. 부상 이력별 가산
13. 행동·교육 메시지
```

# 1단계 · 안전 게이트

## L02 — 50세 이상은 보수적 처방
- 발동: 나이 50 이상
- 동작: 주 빈도 2~3, 렙 8~12, 세트 2~3, %1RM 60~80, 단백질 1.2~1.6 g/kg
  + "첫 8주 적응기" 경고 추가
- 출처: ACSM 2026 / ESPEN ★★★
- 결정거리: 50세 컷오프가 맞나(60·65세 다른 디테일 가이드도 있음)

## L03 — 18세 미만은 1RM 테스트 잠금
- 발동: 나이 18 미만
- 동작: 1RM 측정 불가, RPE 기반 강도, 렙 6~15, "성장판 안전·폼 우선·코치 감독 필수" 경고
- 출처: NSCA Youth Position Statement ★★★
- 결정거리: 미성년자 가입 정책 자체가 있는가? 약관과 충돌하지 않는지

## L05 — 당뇨 환자 처방
- 발동: 질환에 "당뇨"
- 동작: 유산소 3~7회/주(연속 비활동 ≤ 2일), 저항 2~3회, 저항 RPE 상한 7, 유산소 75~150분/주, 8~10종목 1~3세트, "식후 30분 가벼운 걷기" 권고
- 회피 조건 메시지: 망막증·자율신경병증·공복혈당>250+케톤
- 출처: ACSM 2022 + ADA 2024 + 대한당뇨병학회 2023 ★★★

## L06 — 고혈압 환자 처방
- 발동: 질환에 "고혈압"
- 동작: 유산소 5~7회, 저항 2~3회, 저항 RPE 상한 6, %1RM 60~70, 렙 8~12, Valsalva 금지, 유산소 90~150분/주
- **즉시 차단 조건**: 안정시 SBP ≥ 180 또는 DBP ≥ 110 → 처방 중단, 내과 의뢰
- 출처: ACSM 2022 + 대한고혈압학회 2022 ★★★
- 결정거리: 안정시 혈압을 온보딩에서 입력받을 건지(미입력자는 차단 조건 미발동)

## L07 — 이상지질혈증 처방
- 발동: 질환에 "이상지질혈증"
- 동작: 유산소 5~7회, 저항 2~3회, 유연성 2~3회, RPE 상한 6, 유산소 200~300분/주(최소 150), 주당 1,000~2,000 kcal 소비
- 추가: LDL > 190이면 의료 의뢰 메시지 추가
- 출처: ACSM 2022 + 한국지질학회 2022 ★★★

## L08 — 골관절염(무릎·고관절) 처방
- 발동: 질환에 "골관절염_무릎" 또는 "골관절염_고관절"
- 동작: 저항 2~3회, 유산소 3~5회, 유연성 매일, RPE 5~7, 통증 NRS 4 이내, %1RM 60~70, 렙 8~15
- 무릎 OA → 무릎 굽힘 90° 이내
- 회피 동작: 깊은 스쿼트·점프·착지·중량 런지·갑작스런 방향전환
- 권장: 수중운동·태극권
- 출처: OARSI 2019 + ACR 2019 ★★★

# 3단계 · 영양 (BMR → TDEE → 목표 칼로리)

## K19 vs K20 — BMR 공식 자동 분기
- 발동: 항상
- 동작:
  - 체지방률 입력 안 함 → K19 (Mifflin-St Jeor)
    - 남: `10×체중 + 6.25×키 − 5×나이 + 5`
    - 여: 위 식 −161
  - 체지방률 입력함 → K20 (Katch-McArdle)
    - `370 + 21.6 × (체중 × (1 − 체지방률/100))`
- 예시: 남 70kg, 175cm, 30세, 체지방률 미입력 → K19 → 1,649 kcal
- 출처: Mifflin 1990 ★★★ / Aragon ISSN 2017 ★★★

## K22 — 활동 카테고리 자동 분류
- 발동: 직업 + 주당 운동 빈도 입력
- 동작:
  - 육체노동 + 주3~5회↑ 운동 → 매우 활발
  - 주6~7회 운동 → 활발
  - 주3~5회 → 보통
  - 주1~3회 → 가벼움
  - 그 외 → 좌식
- 출처: FAO/WHO/UNU 2001 ★★☆

## K21 — TDEE 계산
- 동작: TDEE = BMR × PAL
  - 좌식 1.2 / 가벼움 1.375 / 보통 1.55 / 활발 1.725 / 매우 활발 1.9
- 예시: BMR 1,649 + "가벼움" → 1,649 × 1.375 ≈ 2,267 kcal
- 출처: FAO/WHO/UNU 2001 ★★★

## 목표 칼로리 가산 (룰 외, 진입 함수에 박힌 로직)
- 다이어트 −500 / 벌크업 +300 / 근육량 증가 +200 / 그 외 0
- 결정거리: 가산 폭이 적정한지(다이어트 -500이 너무 빡센 경우 있음)

# 4단계 · 분할/프로그램 선택

## D05 — 분할 형태
- 동작:
  - 가용일 ≤ 3 또는 초보 → "풀바디 주N"
  - 가용일 4 → "상하분할 주4"
  - 가용일 5↑ → "PPL 주N"
- 출처: Schoenfeld 2016 / Israetel ★★★

## 프로그램 선택 분기 (D06/D07/D08/D10 + 폴백)
- 초보 → D06 Starting Strength
- 스트렝스 + 중급 → D07 5/3/1 Wendler
- 스트렝스 + 그 외 → D08 GZCLP
- 벌크업/근육량 증가 + 중급 + 가용일 ≥ 4 → D10 PHUL/PHAT
- 그 외 모두 → D06 (폴백)

## D06 — Starting Strength
- 출력: A루틴(스쿼트·벤치·데드) / B루틴(스쿼트·OHP·파워클린 또는 바벨로우), 주 3회, 세션당 +2.5~5kg, 6~12개월 운영
- 출처: Rippetoe ★★★

## D07 — 5/3/1 Wendler
- 출력: 트레이닝 맥스 = 1RM × 0.9
  - 1주: 5렙×3 (65/75/85% TM)
  - 2주: 3렙×3 (70/80/90)
  - 3주: 5/3/1+ AMRAP (75/85/95)
  - 4주: 디로드 (40/50/60)
- 출처: Wendler ★★★

## D08 — GZCLP
- 출력: T1(메인 5+렙 탑셋) / T2(보조 10+렙) / T3(아이솔 15+렙), 주 4회
- 출처: r/Fitness wiki ★★☆

## D10 — PHUL/PHAT
- 동작: 가용일 ≤ 4 → PHUL / 5↑ → PHAT, 근력일+근비대일 결합
- 출처: Norton / Campbell ★★☆

## (호출 안 됨)
- D01 Linear Periodization, D03 Block, D09 nSuns, D04 Flexible DUP — 코드는 있는데 분기에서 안 뽑힘
- 결정거리: D01·D03 어떤 케이스에서 띄울지(현재는 D02만 중급에 박혀 있음, 7단계에서)

# 5단계 · 시작 무게

## 분기 1: 중급 이상 + 1RM 입력함
- 시작 무게 = 1RM × 0.8 (운동마다)
- 결정거리: 0.8 고정이 맞나? 종목별로 다른가?

## D11 — 남성 초보 시작 무게
- 출력: 스쿼트 20·벤치 20·OHP 20·데드 40 (kg, 빈 바벨 기준)
- 출처: StrongLifts 5x5 ★★☆

## D12 — 여성 초보 시작 무게
- 동작: 헬스장에 15kg 여성용 바 있으면 바=15kg, 없으면 20kg, 데드 35kg
- 출처: IPF Technical Rules 2024 ★★★
- 결정거리: 사용자가 "헬스장 여성 바 유무"를 어떻게 알지? 디폴트는 어느 쪽?

## D14 — 첫 세션 무게 상한 클램프
- 동작: 종목별 캡(스쿼트 65/35, 데드 70/40, 벤치 40/25, OHP 30/18) × (체중/기준체중)
  - 기준체중 남 70kg, 여 55kg
- 사용자가 더 높게 입력했어도 캡까지 떨굼 + 메모
- 예시: 남 80kg가 첫 스쿼트 100kg 요청 → 클램프 후 약 74kg, "첫 세션 상한" 메모
- 출처: Rippetoe ★★☆

## D15 — 증량 폭 (초기)
- 동작:
  - 하체(스쿼트·데드): 초기 +10kg, 이후 +5kg
  - 상체(벤치·OHP·로우): 초기 +5kg, 이후 +2.5kg
- 진입에선 "초기 폭"만 사용. 이후 폭은 미구현
- 출처: Rippetoe ★★☆

## C06 — 근육군당 최소 빈도
- 출력: 주 2회 (고정)

## C07 — 휴식 시간 (목표별)

## B02 — 시작 볼륨 (근육군별 세트/주)
- 출력: 가슴 8 / 등 10 / 어깨 8 / 이두 8 / 삼두 6 / 다리 7 / 둔근·햄 6 / 복근 0 / 기타 8
- 진입은 6대 근육군(가슴·등·어깨·다리·이두·삼두)에 일괄 적용

- 결정거리: 둔근·햄·복근·전완은 처방 안 됨, 어디 단계에서 넣을지

## B05 — 권장 주간 볼륨 범위

  - 벌크업·근육량 증가 → 12~20세트

  - 다이어트 → 10~18
  - 건강증진 → 8~12

# 7단계 · 주기화/디로드

## 레벨별 주기화 모델
- 초보 → I01 (Linear Progression)
- 중급 → D02 (DUP)
- 고급 → D03 (Block)

## D02 — DUP (중급)
- 출력: 월 비대(8~12렙, 67~75%) / 수 근력(3~6렙, 85~92%) / 금 파워(1~3렙, 90~100%)
- 출처: Painter 2012 / Colquhoun 2017 ★★★

## D03 — Block (고급)
- 출력: 축적(1~4주, 볼륨↑) → 변환(5~7주, 강도↑) → 실현(8~10주, 최대 강도)
- 출처: Issurin / Verkhoshansky ★★☆

## G01 — 디로드 정의
- 출력: 디로드 5~7일, 주기 4~6주
- 출처: Bell / Frontiers ★★★

## G10 — 한국 사용자 디로드 주기
- 동작:
  - 초보 → 신호 기반 자동 (정해진 주기 없음)
  - 중급 이상 → 강제, 6주마다
- 진입에선 "5주 훈련 + 1주 디로드 = 메조 6주"로 박힘
- 출처: RP 메조 가이드 ★★☆

## G05 — 디로드 정량 처방
- 출력: 세트 절반, 무게 ×0.95, 7일
- 출처: Helms 3DMJ ★★★

- D01·D04·G03·G04·G06·G08·G09·G11
- 결정거리: G04(주간 +10% 부하 검사)·G07(다지표) 같은 트리거를 어디 단계에서 평가할지(현재 추천에서는 미발동)

# 8단계 · 운동 선택

## J05 — 컴파운드:아이솔 비율
- 동작: 초보 70:30 / 중급 60:40 / 고급 50:50
- 출처: Israetel / NSCA ★★☆

## J03 — 자유중량:머신 비율
- 동작:
  - 초보 또는 (당뇨·고혈압) → 40:60 (머신 비중↑)
  - 스트렝스 → 80:20
  - 그 외 → 60:40
- 출처: Schwanbeck 2020 ★★★

## J02 — 장비 환경에 맞는 운동 매칭
- 동작: 사용자 장비에 "바벨" 있으면 원래 운동 / 없으면 덤벨→케틀벨→밴드→맨몸 순서로 대체
- 예시: 바벨 없고 덤벨만 → 백스쿼트 → 덤벨고블릿스쿼트
- 출처: Free Exercise DB ★★★

## I03 — 운동 순서 (컴파운드 우선)
- 동작: 가중치 표(빅3=1, 보조 컴파운드=2, 머신 컴파운드=4, 아이솔=5)로 정렬
- 출처: NSCA Essentials ★★★

## J01 — 부상별 대체 운동 매트릭스
- 발동: 부상 이력 부위별
- 매핑:
  - 무릎 → 백스쿼트 → 레그프레스 / 벨트스쿼트 / 스플릿스쿼트
  - 어깨 → 벤치프레스 → 푸시업 / 덤벨프레스 / 랜드마인프레스
  - 허리 → 데드리프트 → 힙쓰러스트 / RDL / 백 익스텐션
  - 손목 → 백스쿼트 → safety bar 스쿼트 / 프론트스쿼트
  - 팔꿈치 → 벤치프레스 → 체스트프레스 머신 / 덤벨플라이
- 출처: NASM / Israetel ★★☆
- 결정거리: 매트릭스 커버 범위가 좁음(다리·가슴 외엔 대체 운동 빈약)

- 결정거리: J07(근육군 매핑) 안 쓰면 근육군별 볼륨 카운트가 부정확함 — 추적 단계에서 필요

# 9단계 · 워밍업/스트레칭

## I05 — 일반 워밍업
- 동작: `min(10, max(5, 세션가용분/6))` 분 저강도 유산소
- 예시: 세션 60분 → 10분 워밍업
- 출처: NSCA Dynamic Warm-Up ★★★

## I06 — 컴파운드 워크업
- 발동: 시작 무게에 "스쿼트" 있을 때
- 출력 (스쿼트 무게 기준):
  - 빈 봉 5렙 × 2세트
  - 40% × 5렙 / 60% × 3렙 / 80% × 2렙
  - 본 세트
- 결정거리: 다른 컴파운드(벤치·데드·OHP)는 워크업 미발동 — 모두에 적용해야

## P01 — 워밍업 스트레칭 정책
- 출력: 정적 회피 / 동적 권장 / 정적은 운동 후
- 출처: Behm 2016 ★★★

## P05 — 정적 스트레칭 경고 (메시지)
- 출력: "워밍업 정적 스트레칭 ≥ 45초 회피 — 최대 근력 평균 -5.4%"
- 출처: Simic 2013 ★★★

## P03 — SMR (텍스트만)
- 출력: "폼롤러 30~60초/부위"
- 출처: Wiewelhove 2019 ★★★

## P04 — 모빌리티 디폴트
- 출력: 고관절 3분 + 발목 2분 + 흉추 3분
- 출처: Page 2012 ★★☆

# 10단계 · 카디오

## O01 — 간섭 검사
- 동작: 진입에선 다이어트·건강증진 → 주 2회 / 그 외 → 주 1회, 분=30 고정
- 출력: 안전(주 < 3회 또는 ≤ 30분) 여부 + "근력 먼저 또는 6시간↑ 분리" 안내
- 출처: Wilson 2012 ★★★

## O02 — HIIT vs LISS 선택
- 동작:
  - 가용분 < 20 → "HIIT 15분"
  - 근력 빈도 ≥ 4 → "LISS 30분"
  - 그 외 → "HIIT 1~2 + LISS 1~2 혼합"
- 출처: Schoenfeld / Helms ★★★

## O03 — 카디오 타이밍
- 동작: 진입에선 strength_focus=True, leg_day=False 고정
- 출력: "근력 후 카디오 또는 별도 세션"
- 출처: Robineau 2016 ★★☆
- 결정거리: leg_day 입력 받아 다리 운동 후 러닝 회피 권고를 동적으로 띄울지

# 12단계 · 부상 이력별 가산

> **공통 호출**: 진입에선 days_since_onset=30, pain_nprs=2 으로 하드코딩되어 있음 → 모두 "복귀 단계"로만 떨어짐. **사용자 입력 필드 매핑이 빠져 있음.**

## Q04 — 요추 통증 프로토콜
- 발동: 부상 이력에 "허리"
- 분기:
  - 적색기 신호(다리 양측 저림·안장마비·괄약근 이상·발열·암 병력 등) → 차단, 응급실/신경외과
  - < 3일 → 일상 활동 + 골반 틸트 / 데드·스쿼트 회피
  - 3~14일 → 스트레칭·근력·유산소·운동조절
  - ≥ 14일 → 글루트·코어 활성화, 데드·스쿼트는 1RM 30%부터, 주 +10% 증가
- 출처: NICE NG59 + Foster 2018 ★★★

## Q05 — 어깨 RCRSP 프로토콜
- 발동: 부상 이력에 "어깨"
- 분기:
  - 적색기(외상 후 거상 불가·진행성 위약·야간통+종괴·발열) → 차단, 정형외과
  - < 7일 → 오버헤드 회피, 진자운동
  - 7~42일 → 밴드 외회전·서라투스 펀치·로우, NPRS 상한 4
  - ≥ 42일 → 벤치·OHP·풀업 1RM 30%부터
- 출처: Lewis 2016 / Cools 2015 ★★★

## Q06 — 무릎 PFPS 프로토콜
- 발동: 부상 이력에 "무릎"
- 분기:
  - 적색기(외상 후 부종·관절잠김·체중부하 불가) → 차단, 정형외과
  - < 14일 → 딥 스쿼트·런지·오픈체인 레그익스텐션 회피
  - 14~42일 → 클램쉘·힙 어브덕션·미니 스쿼트 0~45°
  - ≥ 42일 → 스쿼트·런지 점진
- 출처: Powers 2017 / Crossley 2016 ★★★

## (호출 안 됨)
- Q02 부상 복귀 단계 전환 / Q03 복귀 기준 4가지 충족
- 결정거리: 부상 발생 시점·통증 점수를 사용자 입력으로 어떻게 받을지(현재 30일/2점 하드코딩)

# 13단계 · 행동·교육

## R10 — 한국 직장인 회식 대응
- 발동: 진입에서 "오늘 회식 = False" 고정 호출
- 출력: "오늘 정상 진행, 주간 볼륨 유지"
- (회식일 때): 휴식 권고 + 익일 20분 압축 루틴 + 주 볼륨 -10~15%
- 출처: KOSIS + Parr 2014 ★☆☆
- 결정거리: 사용자 회식 일정을 어떻게 입력받지? 매일 묻기? 캘린더 연동?

## N02 — 신경 적응 vs 근비대 기대값 메시지
- 동작:
  - 훈련 < 2개월 → "초기 4~8주는 신경 적응, 사이즈 변화 작음"
  - 2~8개월 → "8주부터 본격적 근비대"
  - 8개월↑ → "고급 단계, 누적은 크지만 진보 느림"
- 출처: Sale 1988 / Counts 2017 ★★★

## N04 — 근섬유 타입 교육
- 출력: "근비대는 6~30렙 모두 효과적. 저렙=속근만 자극은 단순화 오류"
- 출처: Schoenfeld 2021 ★★★

## M03 — 1RM 측정 방식
- 동작:
  - 초보 또는 60세↑ → "AMRAP로 추정만"
  - 중급 → "AMRAP 8주마다"
  - 고급 → "1RM 직접 테스트 옵션"
- 출처: Reynolds 2006 / NSCA ★★★
