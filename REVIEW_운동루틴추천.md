# 운동 루틴 추천 룰 — 한 줄씩 같이 읽는 회의 문서

## 이 함수는 언제 호출되나
사용자가 온보딩에서 프로필을 완성했을 때 1회. 받아오는 건 사용자 한 사람 정보(성별·체중·키·나이·훈련 개월·목표·주가용일·세션 가용분·체지방률·1RM·부상이력·질환·혈압·LDL·장비). 돌려주는 건 주간 프로그램 한 묶음(BMR·TDEE·목표 칼로리·분할 형태·추천 프로그램·주간 스케줄·시작 무게·근육군별 주간 세트·디로드 주기·운동 풀·워밍업·카디오·회복 가이드·교육 메시지).

## 흐름
13단계를 순서대로. 1단계에서 차단 나오면 처방 정지하고 의료 의뢰만 띄움.

```
1.  안전 게이트 (50세↑·청소년·당뇨·고혈압·이상지질혈증·골관절염)
2.  레벨 분류 (초보·중급·고급)
3.  영양 (BMR → TDEE → 목표 칼로리)
4.  분할/프로그램 선택
5.  시작 무게 + 증량 폭
6.  강도/볼륨
7.  주기화/디로드
8.  운동 선택
9.  워밍업/스트레칭
10. 카디오
11. 회복/수면
12. 부상 이력별 가산
13. 행동·교육 메시지
```

# 사용자 목표 5가지가 흐름을 어떻게 바꾸나

UserProfile.목표는 다음 5개 중 하나 (`types.py:158`):
**다이어트 / 벌크업 / 스트렝스 / 근육량 증가 / 건강증진**

| 목표 | 영양 (3단계) | 강도/볼륨 (6단계) | 휴식 (6단계) | 프로그램 (4단계) | 카디오 (10단계) |
|---|---|---|---|---|---|
| 다이어트 | TDEE -500 | 8–15렙 / 60–80% / 권장 10–18세트 | 컴 150·아이 90 | 폴백 D06 | 주 2회 30분 |
| 벌크업 | TDEE +300 | 6–12렙 / 67–85% / 12–20세트 | 컴 150·아이 90 | 중급+가용4↑ → D10 PHUL/PHAT, 그 외 D06 | 주 1회 30분 |
| 스트렝스 | 변동 없음 | 1–6렙 / 85–100% / 8–16세트 / 자유중량 80:머신 20 | 컴 240·아이 180 | 중급 → D07 5/3/1, 그 외 → D08 GZCLP | 주 1회 30분 |
| 근육량 증가 | TDEE +200 | 6–12렙 / 67–85% / 12–20세트 | 컴 150·아이 90 | 중급+가용4↑ → D10, 그 외 D06 | 주 1회 30분 |
| 건강증진 | 변동 없음 | 12–20렙 / 50–67% / 8–12세트 | 컴 150·아이 90 | 폴백 D06 | 주 2회 30분 |

**목표 무관 (어느 목표여도 동일하게 처방되는 단계)**:
- 1단계 안전 게이트 — 질환·나이만 봄
- 2단계 레벨 분류 — 훈련 개월만 봄
- 3단계 BMR/TDEE 계산식 — 체중·키·나이·체지방률만. 목표는 마지막 가산만
- 5단계 시작 무게 — 성별·체중·1RM
- 7단계 주기화 — 레벨로만 분기 (DUP·Block)
- 8단계 운동 선택 — 레벨·장비
- 9단계 워밍업 — 가용시간
- 11단계 회복/수면, 12단계 부상이력, 13단계 행동·교육

**회의 결정거리**:
1. 목표 5개로 충분한가? (예: "재활"·"기능성"·"운동선수"·"체중유지" 같은 추가 목표 필요?)
2. "벌크업"과 "근육량 증가"의 차이를 사용자가 구분 가능한가? 처방상 차이가 영양 가산 +300 vs +200 뿐이라 사실상 같음
3. 카디오가 다이어트·건강증진만 주2회, 벌크업/스트렝스/근육량 증가는 주1회 — 의도적인 분기인지 확인
4. 스트렝스만 휴식이 길고(컴 240) 자유중량 비중↑(80:20) — 다른 목표와 강하게 갈라짐. 맞음?

# 1단계 · 안전 게이트

## L02 — 50세 이상은 보수적 처방

**한 줄 요약**: 50세 이상 사용자는 노화로 근감소(sarcopenia)가 진행되므로 저항운동을 주 2~3회로 묶고 강도는 60~80% 1RM 안에서, 첫 8주를 적응기로 빼서 무리하지 않게 출발시킨다.

**언제 발동되나**
- 사용자 나이 50세 이상이면 자동 발동 (다른 질환 입력 없어도)
- 처방 전 단계에서 1회 분기

**처방 내용**
- 저항 운동 — 주 2~3회
- 렙수: 8~12렙
- 세트: 2~3세트
- 강도: %1RM 60~80
- 단백질: 1.2~1.6 g/kg/일 (50세 이상 노인의 단백질 요구량은 일반 성인보다 높게 잡음 — ESPEN 권고)
- 메시지: "첫 8주는 적응기" 안내 추가 — 갑자기 평소 무게로 들어가지 말 것

**예시 시나리오**
- 입력: 55세 남, 훈련 경험 없음, 주 4일 가용
  - 시스템 출력: 주 3회 / 8~12렙 / 2~3세트 / %1RM 60~80
  - 첫 8주 적응기 메시지 동시 표시
- (다른 케이스) 같은 사용자가 60세, 부상 이력 무릎 추가
  - L02(노인) + 12단계 부상 처리(Q06)가 둘 다 발동
  - 무릎 굽힘 ROM 제한이 더해짐

**출처**
ACSM 2026 / ESPEN ★★★

**회의 결정거리**
- 50세 컷오프가 맞나(60·65세 다른 디테일 가이드도 있음)
  - 옵션 A: 50세 단일 컷오프 유지 (현재)
  - 옵션 B: 50/65세 2단 컷오프 (65세↑은 더 보수적)
  - 옵션 C: 50/65/75세 3단 분기

## L03 — 18세 미만은 1RM 테스트 잠금

**한 줄 요약**: 18세 미만 청소년은 성장판이 아직 닫히지 않아 1RM(최대중량) 직접 측정을 잠그고, 강도는 RPE(자가 인지 강도)로만 처방한다.

**이게 뭐냐면**
- 1RM = 한 번에 들 수 있는 최대 무게. 측정 중 폼 무너짐과 과부하가 흔해 청소년에게 위험
- RPE(Rate of Perceived Exertion) = "한 세트 끝났을 때 더 할 수 있는 횟수"로 강도를 표현. 절대 무게 대신 자가 인지로 강도 관리
- NSCA Youth Position Statement: 청소년 저항훈련은 폼·코치 감독·점진성이 핵심. 1RM 직접 측정은 권장하지 않음

**언제 발동되나**
- 사용자 나이 18세 미만이면 자동 발동
- 처방 전 단계에서 1회 분기

**처방 내용**
- 1RM 직접 측정 잠금 — 입력 칸에서 1RM 항목 비활성화
- 강도 처방은 RPE 기반으로만
- 렙수: 6~15렙 (저렙 너무 무거운 영역 회피)
- 메시지: "성장판 안전·폼 우선·코치 감독 필수" 경고 노출

**예시 시나리오**
- 입력: 16세 남, 훈련 1년차
  - 시스템 출력: 1RM 입력 칸 잠금 + RPE 기반 처방 (예: 8렙 RPE 7)
  - "코치 감독 필수" 안내 메시지

**출처**
NSCA Youth Position Statement ★★★

**회의 결정거리**
- 미성년자 가입 정책 자체가 있는가? 약관과 충돌하지 않는지
  - 옵션 A: 18세 미만 가입 자체 차단
  - 옵션 B: 가입 허용하되 보호자 동의 받고 L03 자동 적용 (현재 룰 가정)
  - 옵션 C: 16~17세는 허용, 16세 미만 차단

## L05 — 당뇨 환자 처방

**한 줄 요약**: 당뇨 환자는 인슐린 감수성을 높이고 혈당을 안정시키기 위해 유산소를 주 3~7일로 자주 가져가고, 저항운동은 주 2~3일·RPE 7 이내로 보수적으로 깐다. 합병증(망막증·자율신경병증·고혈당+케톤)이 있으면 처방을 멈추고 의료 의뢰로 돌린다.

**이게 뭐냐면**
- 제2형 당뇨에서 운동의 1차 효과는 "인슐린 감수성 개선" — 근육이 포도당을 더 잘 끌어다 쓰게 됨
- 운동의 혈당 강하 효과는 24~48시간 정도만 지속 → "이틀 연속 비활동 회피"가 핵심 권고
- Valsalva(호흡 참고 들어올리기)는 일시적 혈압 급등을 일으켜 망막증·자율신경병증 환자에게 위험
- SU계 = 설포닐우레아계 경구 혈당강하제(글리메피리드·글리피지드 등). 인슐린과 함께 운동 중 저혈당 위험 약물

**언제 발동되나**
- 사용자가 질환 항목에 "당뇨"를 체크하면 자동 발동
- 1형/2형 구분 없이 진입 함수는 동일 처방
- 사용자 프로필 등록 시·강도 처방 시 모두 호출

**필요한 입력**
- 진단명(제2형 당뇨)
- 복용 약물(인슐린·SU·메트포르민 등)
- 합병증(망막증/신증/신경병증/족부)
- HbA1c·공복혈당
- 운동 가능 시간

**처방 내용**
- 유산소 운동 — 주 3~7회
  - 연속 이틀 이상 쉬지 말 것 (혈당 강하 효과가 24~48시간만 지속)
  - 시간: 주 합계 75~150분
- 저항 운동 — 주 2~3회
  - RPE 상한 7 ("2~3개 더 할 수 있겠다" 정도)
  - 8~10종목 × 1~3세트, 전신 골고루
- 식후 30분 가벼운 걷기 권고 메시지
  - 식후 혈당 스파이크 감소 효과 (Diabetes Care 36(10):3262 등)

**처방 안 하고 의료 의뢰로 돌리는 조건** (이 중 하나라도 해당)
- 망막증 진단 — Valsalva·점프·머리 숙임이 망막박리 위험을 키움 → 안과 의뢰
- 자율신경병증 — 심박수·혈압 반응이 비정상 → 일반 강도 처방 위험
- 공복혈당 > 250 mg/dL + 케톤(+) — 운동 시 케톤산증 악화 가능 → 운동 보류
- 저혈당 빈발(주 2회 이상) 또는 무자각성 저혈당 → 내분비내과 상담

**예시 시나리오**
- 입력: 38세 남, 2형 당뇨 5년차, 합병증 없음, 주 4일 가용, 60분/세션
  - 시스템 출력:
    - 유산소: 주 4회 × 25분 (걷기·자전거)
    - 저항: 주 2회 × 8종목 × 2세트, RPE ≤ 7
  - 메시지: "식후 30분 가볍게 10분 걷기 추가 권장"
- (다른 케이스) 같은 사용자 + 망막증 추가 진단
  - 처방 중단 → "안과 진료 후 재방문" 메시지
- (인슐린 복용자) 운동 전 혈당 < 100 mg/dL 입력
  - "탄수 15g(바나나 1/2개) 보충 후 시작" 안내

**출처**
ACSM Guidelines for Exercise Testing and Prescription 11th ed (2022) + ADA Standards of Care 2024 + 대한당뇨병학회 2023 진료지침 ★★★

**회의 결정거리**
- 합병증을 어떻게 입력받을 건가?
  - 옵션 A: 온보딩 체크리스트로 자가 보고 (가장 간단)
  - 옵션 B: 진단명 코드(KCD) 입력 (정확하지만 진입장벽↑)
  - 옵션 C: "병원에서 합병증 있다고 들었다" 단순 Y/N
- 공복혈당·케톤 — 매 세션 vs 주 1회 vs 입력 안 받기?
  - 옵션 A: 매 세션 (안전하지만 사용성↓)
  - 옵션 B: 주 1회 갱신
  - 옵션 C: 처음 1회 입력 후 유지

## L06 — 고혈압 환자 처방

**한 줄 요약**: 고혈압 환자는 유산소 빈도를 주 5~7회로 높이고, 저항운동은 RPE 6 이내·1RM 60~70%로 보수적으로 깐다. Valsalva(호흡 참기) 절대 금지. 안정시 혈압 180/110 이상이면 처방 자체를 중단하고 내과로 보낸다.

**이게 뭐냐면**
- 운동 중 Valsalva = 호흡을 참고 복압을 끌어올려 들어올리는 기법. 코어 안정성에 유리하지만 일시적으로 수축기 혈압을 200mmHg 이상으로 올림
- 베타차단제 = 심박수를 억제하는 고혈압 약. 복용자는 운동 중 심박수가 둔해져 HR 기반 강도 관리가 부정확 → RPE로 강도 관리
- 알파차단제 = 운동 후 기립성 저혈압 위험 → 쿨다운 5분 필수

**언제 발동되나**
- 사용자 질환에 "고혈압" 체크 시 자동 발동
- 사용자 프로필 등록 시·강도 처방 시 모두 호출

**필요한 입력**
- 진단명(고혈압 1·2기)
- 복용 약물(β-차단제·이뇨제·ACEi·ARB·CCB)
- 안정시 BP 최근치
- 동반질환(CVD/당뇨)

**처방 내용**
- 유산소 운동 — 주 5~7회 (가능하면 매일)
  - 시간: 주 90~150분, 30분/일 또는 누적
  - 형태: 걷기·자전거·수영 등 유산소 우선
- 저항 운동 — 주 2~3회
  - RPE 상한 6 (더 보수적)
  - %1RM 60~70 / 8~12렙 / 2~4세트
  - 머신 비중↑ (프리웨이트 고중량 회피)
- 호흡: Valsalva 절대 금지 — 들어올릴 때 호기, 내릴 때 흡기
- 베타차단제 복용자: HR 대신 RPE로 강도 관리

**즉시 차단 / 의료 의뢰 조건**
- 안정시 BP ≥ 180/110 mmHg → 처방 중단, 내과 의뢰
- 미조절 고혈압(BP > 160/100 지속) → 심장내과 상담 (차단은 X)
- 운동 중 가슴통증·심한 두통·시야 흐림·호흡곤란·실신감 → 즉시 중단·119
- 운동 중 SBP > 250 또는 DBP > 115 → 즉시 중단

**예시 시나리오**
- 입력: 50세 여, 고혈압 1기, 베타차단제 복용, 안정시 BP 145/90, 주 4일 가용
  - 시스템 출력:
    - 유산소: 주 5회 × 30분
    - 저항: 주 2회, %1RM 60~70, RPE ≤ 6
  - 메시지: "심박수 대신 RPE로 강도 가늠. Valsalva 금지"
- (다른 케이스) 같은 사용자 + 안정시 BP 185/95
  - 처방 중단 → "내과 진료 후 재방문" 메시지

**출처**
ACSM Guidelines 11th ed (2022) + ACSM Pronouncement on Exercise & Hypertension 2023 + 대한고혈압학회 2022 진료지침 ★★★

**회의 결정거리**
- 안정시 혈압을 온보딩에서 입력받을 건지(미입력자는 차단 조건 미발동)
  - 옵션 A: 미입력 시 처방 보수 모드 (RPE 5 이내, 저강도 유산소만)
  - 옵션 B: 미입력 시 처방 진행하되 "최근 측정값 알려달라" 메시지 노출
  - 옵션 C: 고혈압 체크 시 BP 입력 강제

## L07 — 이상지질혈증 처방

**한 줄 요약**: 이상지질혈증 환자는 지질 개선 효과가 큰 유산소를 길게 가져간다. 주 200~300분(최소 150분), 칼로리 소비 1,000~2,000 kcal/주가 최적. LDL > 190이면 운동 시작 전 의료 평가가 필요함을 알린다.

**이게 뭐냐면**
- 이상지질혈증 = 혈중 지질(LDL·중성지방·HDL) 이상. 운동의 지질 개선 효과는 강도보다 "총 시간/칼로리 소비"에 더 의존
- LDL > 190 mg/dL은 가족성 고콜레스테롤 의심선. 자체로 운동부하검사 권장 사유
- 한국지질·동맥경화학회 5판: 주 200~300분이 지질 개선 최적 구간

**언제 발동되나**
- 사용자 질환에 "이상지질혈증" 체크 시 자동 발동
- 사용자 프로필 등록 시·강도 처방 시 모두 호출

**필요한 입력**
- 진단명(이상지질혈증·대사증후군)
- 지질 패널(LDL/HDL/TG)
- 복용 약물(스타틴·에제티미브·페노피브레이트)
- 동반질환
- BMI/허리둘레

**처방 내용**
- 유산소 운동 — 주 5~7회
  - 시간: 주 200~300분 (최적), 최소 150분
  - 강도: 중강도 RPE 4~6 (HRmax 55~75%)
- 저항 운동 — 주 2~3회 (비연속일)
- 유연성 — 주 2~3회
- RPE 상한: 6
- 주당 칼로리 소비: 1,000~2,000 kcal (체중감량/지질개선 최적)
- LDL > 190이면 의료 의뢰 메시지

**즉시 차단 / 의료 의뢰 조건**
- 가족성 고콜레스테롤(LDL > 190)
- 기왕 심혈관질환(MI/PCI/뇌졸중)
- 당뇨 + 50세 이상
- 운동 중 가슴통증·턱/왼팔 방사통·심한 호흡곤란
→ 모두 운동 시작 전 심혈관내과 운동부하검사(EST) 권고

**예시 시나리오**
- 입력: 45세 남, 이상지질혈증, LDL 200, 주 5일 가용
  - 시스템 출력:
    - 유산소: 주 5회 × 50분 (= 250분/주, 권장 범위 안)
    - 저항: 주 2회, RPE ≤ 6
  - 메시지: "LDL 200 — 운동부하검사 권고" 동시 표시
- (다른 케이스) 같은 사용자 + 동반 고혈압
  - L06과 L07 동시 발동, 안전 가드는 더 보수적인 쪽 우선

**출처**
ACSM Guidelines 11th ed (2022) + 한국지질·동맥경화학회 이상지질혈증 진료지침 5판 (2022) ★★★

**회의 결정거리**
- LDL 수치를 사용자가 알고 있을까? 모를 때 디폴트 동작?
  - 옵션 A: "수치 모름" 선택 시 일반 처방으로 진행 (안전 가드만 적용)
  - 옵션 B: "수치 모름" 시 의무적으로 검진 안내 메시지
  - 옵션 C: "이상지질혈증 진단 받았으나 수치 모름"이면 LDL > 190 추정해 의뢰 트리거 발동

## L08 — 골관절염(무릎·고관절) 처방

**한 줄 요약**: 골관절염 환자는 통증 NRS 4 이내·RPE 5~7로 묶고, 깊은 스쿼트·점프·중량 런지처럼 관절 충격이 큰 동작은 회피한다. 무릎 OA는 굽힘 90° 제한. 수중운동·태극권을 적극 권한다(OARSI/ACR 강력 권고).

**이게 뭐냐면**
- 골관절염(OA) = 관절 연골 마모로 통증·부종·기능제한. 무릎/고관절이 가장 흔함
- NRS = Numeric Rating Scale, 0(통증 없음)~10(최악)으로 통증을 자가 보고
- OARSI/ACR 2019 가이드라인: "구조화된 지상(land-based) 운동 프로그램"이 1차 치료. 운동 회피가 아니라 "맞는 운동"을 처방

**언제 발동되나**
- 질환에 "골관절염_무릎" 또는 "골관절염_고관절" 체크 시
- 사용자 프로필 등록 시·강도 처방 시 모두 호출

**필요한 입력**
- 진단명(무릎/고관절 OA)
- 통증 NRS(0~10)
- 발병 부위
- 동반질환(GI·CV)
- 보행/계단 가능 여부
- BMI

**처방 내용**
- 저항 운동 — 주 2~3회 (대퇴사두근·둔근·체간 강화)
  - %1RM 60~70 / 8~15렙 / 1~3세트
  - RPE 5~7
  - 통증 NRS 4 이내 유지
- 유산소 운동 — 주 3~5회 (걷기·자전거·수영)
  - 회당 20~30분 (주 150분 목표)
- 유연성/모빌리티 — 매일
- 무릎 OA → 무릎 굽힘 90° 이내 강제

**회피 동작**
- 깊은 스쿼트(>90도 굴곡)
- 점프·착지
- 중량 런지
- 갑작스런 방향전환

**권장 추가**
- 수중운동 (체중부하↓ — 무릎 OA에 강력 권고)
- 태극권 (ACR 2019에서 강력 권고로 격상)

**즉시 차단 / 정형외과 의뢰 조건**
- 야간통(잠 깨는 통증)
- 관절 잠김(locking)
- 갑작스런 무력감(giving way)
- 관절 발적·발열
- 외상성 부종
- 6주+ 보존치료에도 통증/기능 악화

**예시 시나리오**
- 입력: 58세 여, 무릎 OA, NRS 3, 주 3일 가용
  - 시스템 출력: 백스쿼트 회피, 머신 레그프레스 + 박스 스쿼트(평행) + 수중운동 추가
  - %1RM 60~70 / 10~12렙, RPE 5~7
- (다른 케이스) 같은 사용자 + NRS 6
  - 즉시 강도/ROM 감량 권고, RPE 상한 5로 하향

**출처**
OARSI 2019 (Bannuru) + ACR 2019 (Kolasinski) + ACSM Guidelines 11th ed (2022) + 대한정형외과학회 무릎관절염 자료 ★★★

**회의 결정거리**
- 통증 NRS를 매 세션 입력받을지, 주 1회로 충분할지
  - 옵션 A: 매 세션 입력 (실시간 코칭에서도 같은 항목 활용 가능)
  - 옵션 B: 주 1회 갱신
  - 옵션 C: 통증 변화 시점에만 갱신 (사용자가 누름)

# 2단계 · 레벨 분류

## L04 — 훈련 개월로 레벨 결정

**한 줄 요약**: 훈련 개월수만 보고 초보·중급·고급 3단계로 분류한다. 이후 단계의 거의 모든 분기(프로그램 선택·주기화·시작 무게·운동 비율)가 이 레벨 한 변수에 묶여 있다.

**이게 뭐냐면**
- 초보 = 매 세션 진보 가능한 단계 (LP만으로도 매주 무게가 오름)
- 중급 = 주 단위로 진보를 묶어야 함 (DUP·5/3/1 등 자율조절 시작)
- 고급 = 메조사이클(블록) 단위로 봐야 진보가 보이는 단계

**언제 발동되나**
- 진입할 때 항상. 1단계 안전 게이트 이후 가장 먼저
- 초기 셋업 시 1회

**필요한 입력**
- 훈련 기간 (개월)
- (선택) 1RM 비율

**처방 내용 (분기 기준)**
- 훈련 개월 < 6 → 초보
- 6~24개월 → 중급
- 24개월 이상 → 고급

**예시 시나리오**
- 입력: "훈련 9개월" → 중급 → 이후 단계의 모든 중급 분기 활성화 (D02 DUP, J05 60:40 컴파운드, J03 60:40 자유중량 등)
- (다른 케이스) "훈련 36개월" → 고급 → D03 Block 주기화로 분기

**출처**
Rippetoe (Practical Programming) ★★★

**회의 결정거리**
- "훈련 개월"을 어떻게 정의하지? 휴식기 포함? 자가 보고만 믿어도 되나?
  - 옵션 A: 자가 보고 그대로 사용 (현재 가정)
  - 옵션 B: 1RM 비율 같이 받아서 검증 (예: 체중 대비 스쿼트 1RM이 너무 낮으면 초보로 다운그레이드)
  - 옵션 C: 마지막 6개월 연속 훈련 여부만 묻기

# 3단계 · 영양 (BMR → TDEE → 목표 칼로리)

## K19 vs K20 — BMR 공식 자동 분기

**한 줄 요약**: 사용자가 체지방률을 알면 더 정밀한 공식(K20 Katch-McArdle), 모르면 일반 공식(K19 Mifflin-St Jeor)으로 하루 기초 소모 칼로리를 계산한다. 두 공식 중 하나는 항상 실행된다.

**이게 뭐냐면**
- BMR = "하루 종일 누워만 있어도 살아있기 위해 태우는 칼로리". 호흡·심장박동·뇌·체온 유지 베이스라인
- BMR이 모든 영양 계산의 출발점 (BMR → TDEE → 목표 칼로리)
- Mifflin-St Jeor (1990) = 비만·정상 모두 가장 정확(±10%)한 1차 권장식
- Katch-McArdle = 제지방량(LBM) 기반 → 근육 많은 사용자에서 Mifflin보다 정확

**언제 발동되나**
- 진입 시 항상. 두 공식 중 하나는 무조건 실행
- 온보딩 / 체중 변동 시 재계산

**필요한 입력**
- 성별, 체중(kg), 키(cm), 나이(년) — K19용
- 체지방률(%) — K20용 (있으면 K20 우선)

**분기 규칙**
- 체지방률 미입력 → K19 (Mifflin-St Jeor)
  - 남: `10×체중 + 6.25×키 − 5×나이 + 5`
  - 여: 위 식 − 161
- 체지방률 입력함 → K20 (Katch-McArdle)
  - `BMR = 370 + 21.6 × LBM(kg)`
  - LBM = 체중 × (1 − 체지방률/100)

**왜 두 공식을 쓰나**
- Mifflin은 평균값 — 근육 많은 사람은 과소 계산되기 쉬움
- Katch는 근육량을 직접 쓰니 정확하지만 체지방률 측정값 필요
- "체지방률 알면 정확한 거, 모르면 무난한 거" 자동 선택

**예시 시나리오**
- 입력 1: 남 70kg, 175cm, 30세, 체지방률 미입력
  - 분기: K19 (체지방률 없음)
  - 계산: 10×70 + 6.25×175 − 5×30 + 5 = 700 + 1093.75 − 150 + 5
  - **BMR ≈ 1,649 kcal/일**
- 입력 2: 같은 사용자가 체지방률 18% 추가 입력
  - 분기: K20
  - LBM = 70 × (1 − 0.18) = 57.4 kg
  - 계산: 370 + 21.6 × 57.4 = 370 + 1239.84
  - **BMR ≈ 1,610 kcal/일**

**출처**
Mifflin & St Jeor 1990 (Am J Clin Nutr 51:241-7) ★★★ / Aragon et al. 2017 ISSN Position Stand ★★★

**회의 결정거리**
- 체지방률 입력값의 신뢰도(체성분 측정·인바디·자가 추정)별 분기 필요한지
  - 옵션 A: 그대로 사용 (자가 입력값)
  - 옵션 B: 측정 방법(인바디·캘리퍼·DEXA) 같이 받아서 신뢰도 보정
  - 옵션 C: ±5%p 이상 의심값(예: 6% 자가입력)이면 K19로 폴백

## K22 — 활동 카테고리 자동 분류

**한 줄 요약**: 직업 + 주당 운동 빈도를 받아 좌식·가벼움·보통·활발·매우 활발 5단계 중 하나로 자동 분류한다. 이 카테고리가 다음 단계 K21에서 PAL 계수로 변환된다.

**이게 뭐냐면**
- PAL = Physical Activity Level. BMR을 TDEE로 환산할 때 곱하는 계수
- 한국 직장인 다수가 사무직 + 주 1~3회 운동에 해당 → 보수적인 1.375(가벼움) 권장이 K22의 핵심 설계 의도

**언제 발동되나**
- 직업 + 주당 운동 빈도 입력받은 직후, K21 호출 전
- 온보딩

**필요한 입력**
- 직업 유형
- 주당 운동 빈도
- 운동 강도(RPE)

**분기 규칙 (5단계 매핑)**
- 사무직 + 운동 X → 좌식 (1.2)
- 주 1~3회 헬스 → 가벼움 (1.375)
- 주 3~5회 중강도 → 보통 (1.55)
- 주 6~7회 또는 격렬 → 활발 (1.725)
- 육체노동 + 격한 훈련 → 매우 활발 (1.9)

**예시 시나리오**
- 입력 1: 사무직 + 주 3회 헬스 → 보통 (1.55)
- 입력 2: 사무직 + 운동 X → 좌식 (1.2)
- 입력 3: 육체노동(건설) + 주 5회 격렬 훈련 → 매우 활발 (1.9)

**출처**
FAO/WHO/UNU 2001 PAL 범위 ★★☆

**회의 결정거리**
- 직업 카테고리(사무직·서비스직·육체노동)를 어떻게 받을지
  - 옵션 A: 3분류 라디오 버튼 (단순)
  - 옵션 B: 5분류 (사무직/서비스/판매/제조/건설) + 자가 강도 점수
  - 옵션 C: 통계청 직업분류표 매핑 (정확하지만 무거움)

---

## K21 — TDEE 계산

**한 줄 요약**: BMR에 K22가 정해준 PAL 계수를 곱해 하루 총 에너지 소비(TDEE)를 산출한다. 이 값이 "유지 칼로리"의 기준이 된다.

**이게 뭐냐면**
- TDEE = Total Daily Energy Expenditure, 하루 총 소모 칼로리
- BMR(가만히 있어도 쓰는 칼로리)에 활동량 계수를 곱한 값
- TDEE보다 적게 먹으면 살이 빠지고, 많이 먹으면 늘어남 — 다이어트·벌크업의 출발점

**언제 발동되나**
- 진입 시 항상. K19/K20 → K22 → K21 순으로 호출
- 온보딩 / 활동량 변경 시

**필요한 입력**
- BMR (K19 또는 K20 결과)
- 활동 카테고리 (K22 결과 — 5단계 중 하나)

**처방 내용**
- TDEE = BMR × PAL
- PAL 값:
  - 좌식 1.2
  - 가벼움 1.375
  - 보통 1.55
  - 활발 1.725
  - 매우 활발 1.9

**예시 시나리오**
- 입력: BMR 1,649 + "가벼움"(1.375)
  - 계산: 1,649 × 1.375
  - **TDEE ≈ 2,267 kcal/일**
- (다른 케이스) BMR 1,649 + "보통"(1.55)
  - 계산: 1,649 × 1.55
  - **TDEE ≈ 2,556 kcal/일**

**출처**
FAO/WHO/UNU 2001 Human Energy Requirements ★★★

## 목표 칼로리 가산 (룰 외, 진입 함수에 박힌 로직)

**한 줄 요약**: TDEE에 사용자 목표별로 정해진 칼로리를 더하거나 빼서 최종 목표 칼로리를 만든다. 다이어트 -500, 벌크업 +300, 근육량 증가 +200, 스트렝스·건강증진은 변동 없음.

**언제 발동되나**
- K21 직후 항상 호출
- 진입 함수에 하드코딩된 가산 로직

**처방 내용**
- 다이어트 → TDEE − 500
- 벌크업 → TDEE + 300
- 근육량 증가 → TDEE + 200
- 스트렝스 → TDEE 그대로
- 건강증진 → TDEE 그대로

**예시 시나리오**
- 입력: TDEE 2,267 + 다이어트
  - 계산: 2,267 − 500 = **1,767 kcal/일**
- (다른 케이스) TDEE 2,267 + 벌크업
  - 계산: 2,267 + 300 = **2,567 kcal/일**

**회의 결정거리**
- 가산 폭이 적정한지(다이어트 -500이 너무 빡센 경우 있음)
  - 옵션 A: 고정값 유지 (현재)
  - 옵션 B: 체중 비례 (다이어트 −10kcal/kg 등)
  - 옵션 C: 사용자가 "느슨/표준/공격적" 3단계 직접 선택

# 4단계 · 분할/프로그램 선택

## D05 — 분할 형태

**한 줄 요약**: 가용일과 레벨을 같이 보고 풀바디·상하분할·PPL 중 하나로 자동 매핑한다. 빈도×볼륨이 같다면 분할 형태 자체는 효과 차이가 거의 없다는 메타분석이 근거.

**이게 뭐냐면**
- 풀바디 = 한 세션에 전신을 다 자극하는 분할. 주 2~3회에 적합
- 상하분할 = 상체일/하체일 교대. 주 4회에 적합
- PPL = Push(미는 운동) / Pull(당기는 운동) / Leg(다리). 주 5~6회에 적합

**언제 발동되나**
- 진입 시 항상
- 초기 셋업

**필요한 입력**
- 가용 일수
- 사용자 레벨

**처방 내용**
- 가용일 ≤ 3 또는 초보 → "풀바디 주N"
- 가용일 4 → "상하분할 주4"
- 가용일 ≥ 5 → "PPL 주N"

**예시 시나리오**
- 입력: 중급 + 주 5일 가용 → "PPL 주5"
- 입력: 초보 + 주 5일 가용 → "풀바디 주5" (초보는 가용일 무시하고 풀바디)
- 입력: 중급 + 주 3일 가용 → "풀바디 주3"

**출처**
Schoenfeld 2016 / Israetel ★★★

## 프로그램 선택 분기 (D06/D07/D08/D10 + 폴백)

**한 줄 요약**: 레벨·목표·가용일 조합에 따라 4가지 프로그램(Starting Strength·5/3/1·GZCLP·PHUL/PHAT) 중 하나가 선택되고, 어디에도 안 맞으면 D06으로 폴백한다.

**언제 발동되나**
- D05 분할 결정 직후 항상

**처방 분기 (우선순위 순서)**
- 초보 → D06 Starting Strength
- 스트렝스 + 중급 → D07 5/3/1 Wendler
- 스트렝스 + 초보·고급 → D08 GZCLP
- 벌크업·근육량 증가 + 중급 + 가용일 ≥ 4 → D10 PHUL/PHAT
- 그 외 모두 → D06 (폴백)

**예시 시나리오**
- 중급 + 스트렝스 → D07 5/3/1
- 중급 + 벌크업 + 가용 5일 → D10 PHUL/PHAT
- 고급 + 다이어트 → D06 (폴백)
- 초보 + 어떤 목표든 → D06

**회의 결정거리**
- "고급 + 스트렝스"가 D08 GZCLP에 떨어지는 게 맞나(GZCLP는 보통 초보~중급용)
  - 옵션 A: 고급 + 스트렝스는 D03 Block으로 (현재 D03은 7단계 주기화에서만 호출됨)
  - 옵션 B: 고급 + 스트렝스 → 별도 폴백 D07 5/3/1
  - 옵션 C: 현재 그대로 D08 유지 (고급도 GZCLP 가능)

## D06 — Starting Strength

**한 줄 요약**: 초보용 Linear Progression 프로그램. 빅3 + OHP를 5렙×3세트로 돌리고 매 세션 +2.5~5kg씩 무게를 올린다. 6~12개월 운영 후 졸업.

**이게 뭐냐면**
- LP = Linear Progression, "매 세션 무게 인상"의 가장 단순한 진보 방식
- 초보는 신경 적응 시기라 단순 LP만으로도 매주 늘어남 (N02 참고)
- A·B 루틴을 교대로 — 같은 빅3에 매번 새 자극 변형 추가

**언제 발동되나**
- 위 프로그램 분기에서 D06으로 떨어진 사용자
- 초기 셋업

**필요한 입력**
- 사용자 레벨(초보)
- 1RM (있으면)

**처방 내용**
- A루틴: 스쿼트·벤치·데드
- B루틴: 스쿼트·OHP·파워클린 또는 바벨로우
- 빈도: 주 3회 (A·B·A 또는 B·A·B)
- 증량: 세션당 +2.5~5kg (D15 참고)
- 운영 기간: 6~12개월 (졸업 후 D07/D10로 전환)

**예시 시나리오**
- 월(A): 스쿼트 5×3 + 벤치 5×3 + 데드 5×1
- 수(B): 스쿼트 5×3 + OHP 5×3 + 바벨로우 5×3
- 금(A): 스쿼트 5×3 + 벤치 5×3 + 데드 5×1
- 다음 주: 모든 운동 +2.5kg

**출처**
Mark Rippetoe (Starting Strength) ★★★

## D07 — 5/3/1 Wendler

**한 줄 요약**: 중급 스트렝스 사용자용 4주 사이클 프로그램. 트레이닝 맥스(TM) = 1RM × 0.9를 기준으로 1주차 5렙·2주차 3렙·3주차 5/3/1+ AMRAP·4주차 디로드를 돈다.

**이게 뭐냐면**
- TM(Training Max) = 1RM 직접 사용 시 무리되니 90%로 낮춰서 작업 무게의 기준점으로 사용
- AMRAP = As Many Reps As Possible. 마지막 세트만 한계까지 반복
- 중급은 매 세션 진보가 안 됨 → 4주 사이클로 묶어서 진보 추적

**언제 발동되나**
- 프로그램 분기에서 "중급 + 스트렝스"
- 초기 셋업 / 메조사이클 관리

**필요한 입력**
- 사용자 레벨(중급)
- 1RM

**처방 내용**
- TM = 1RM × 0.9
- 1주차: 5렙 × 3세트 (65 / 75 / 85% TM)
- 2주차: 3렙 × 3세트 (70 / 80 / 90% TM)
- 3주차: 5/3/1+ AMRAP (75 / 85 / 95% TM, 마지막 세트는 한계까지)
- 4주차: 디로드 (40 / 50 / 60% TM)
- 사이클당 TM +2.5~5kg

**예시 시나리오**
- 입력: 1RM 100kg → TM 90kg
  - 1주차 마지막 세트: 90 × 0.85 = 76.5kg × 5렙
  - 2주차 마지막 세트: 90 × 0.90 = 81kg × 3렙
  - 3주차 마지막 세트(AMRAP): 90 × 0.95 = 85.5kg × 1+렙

**출처**
Jim Wendler ★★★

## D08 — GZCLP

**한 줄 요약**: 초·중급 LP 변형. T1(메인 5+렙 탑셋) → T2(보조 10+렙) → T3(아이솔 15+렙) 3-tier 구조로 한 세션을 깐다. 주 4회.

**이게 뭐냐면**
- GZCLP = Cody LeFever의 GZCL 메소드의 LP 버전
- T1/T2/T3 = 메인·보조·아이솔 3계층 구조. 강도와 볼륨을 분리해 처방
- T1은 무겁게, T2는 적당히, T3는 가볍게 많이

**언제 발동되나**
- 프로그램 분기에서 "스트렝스 + 초보·고급"
- 초기 셋업

**필요한 입력**
- 사용자 레벨
- 1RM

**처방 내용**
- T1: 메인 5+렙 탑셋 (가장 무거운 메인 리프트)
- T2: 보조 10+렙
- T3: 아이솔 15+렙
- 빈도: 주 4회

**출처**
Cody LeFever / r/Fitness wiki ★★☆

## D10 — PHUL/PHAT

**한 줄 요약**: 근력일 + 근비대일을 한 주 안에 결합한 하이브리드 분할. 가용일에 따라 PHUL(주4일) 또는 PHAT(주5일)로 분기. 중급 + 벌크업/근육량 증가에 권장.

**이게 뭐냐면**
- PHUL = Power Hypertrophy Upper Lower (Brandon Campbell, 주 4일 상하분할 + 일별 포커스 다름)
- PHAT = Power Hypertrophy Adaptive Training (Layne Norton, 주 5일)
- "근력 + 사이즈" 동시 추구 사용자에게 적합

**언제 발동되나**
- 프로그램 분기에서 "벌크업·근육량 증가 + 중급 + 가용일 ≥ 4"
- 초기 셋업

**필요한 입력**
- 사용자 레벨(중급+)
- 목표(파워+근비대)

**처방 내용**
- 가용일 ≤ 4 → PHUL
- 가용일 ≥ 5 → PHAT
- 구성: 근력일 + 근비대일 결합

**예시 시나리오**
- 중급 + 벌크업 + 주 4일 → PHUL (월=상체 근력, 화=하체 근력, 목=상체 근비대, 금=하체 근비대)
- 중급 + 근육량 증가 + 주 5일 → PHAT

Layne Norton / Brandon Campbell ★★☆

# 5단계 · 시작 무게

## 분기 1: 중급 이상 + 1RM 입력함

**한 줄 요약**: 중급 이상 사용자가 1RM을 입력하면 시작 무게는 1RM × 0.8 (모든 종목 동일 계수)로 자동 계산된다.

**언제 발동되나**
- 레벨 ≥ 중급 AND 1RM 입력값 있음
- 첫 세션 진입 시

**처방 내용**
- 시작 무게 = 1RM × 0.8 (운동마다 동일 계수)

**예시 시나리오**
- 중급 + 스쿼트 1RM 120kg 입력 → 시작 96kg
- 중급 + 벤치 1RM 80kg 입력 → 시작 64kg

**회의 결정거리**
- 0.8 고정이 맞나? 종목별로 다른가?
  - 옵션 A: 0.8 단일 계수 유지 (현재)
  - 옵션 B: 종목별 차등 (스쿼트 0.8 / 벤치 0.75 / 데드 0.7)
  - 옵션 C: 목표별 차등 (스트렝스 0.85 / 벌크업 0.7)

## D11 — 남성 초보 시작 무게

**한 줄 요약**: 남성 초보(또는 1RM 미입력)는 빈 바벨(20kg) 기준으로 스쿼트·벤치·OHP를 시작하고, 데드리프트만 40kg(빈 바 + 양쪽 10kg)으로 시작한다.

**언제 발동되나**
- 남성 + 초보 (또는 1RM 미입력)
- 첫 세션 / 프로그램 진입

**필요한 입력**
- 성별=남
- 운동 경험=없음
- 바벨 종류=20kg 표준

**처방 내용 (kg, 빈 바벨 기준)**
- 스쿼트 20
- 벤치 20
- OHP 20
- 데드 40

**예시 시나리오**
- 남 75kg 초보, 헬스장 첫 방문 → 스쿼트 20kg 5×3 시작

**출처**
StrongLifts 5x5 (Mehdi Hadim) ★★☆

## D12 — 여성 초보 시작 무게

**한 줄 요약**: 여성 초보는 헬스장에 15kg 여성용 바가 있으면 그걸로, 없으면 20kg로 빈 바 운동을 시작한다. 데드리프트는 30~40kg.

**이게 뭐냐면**
- IPF(국제파워리프팅연맹) Technical Rules 2024는 여성 경기용 바를 15kg으로 규정 — 일부 헬스장은 이를 보유
- 빈 바 20kg가 부담스러운 여성 초보에게 더 부드러운 시작점

**언제 발동되나**
- 여성 + 초보 (또는 1RM 미입력)
- 첫 세션 / 프로그램 진입

**필요한 입력**
- 성별=여
- 운동 경험=없음
- 바벨 종류=15kg 또는 20kg

**처방 내용**
- 헬스장에 15kg 여성용 바 있음 → 바벨 운동 = 15kg
- 여성용 바 없음 → 바벨 운동 = 20kg
- 데드: 30~40kg (별도)

**예시 시나리오**
- 여 60kg 초보, 헬스장에 15kg 바 있음 → 스쿼트 15kg 5×3 시작
- 여 55kg 초보, 헬스장에 20kg 바만 있음 → 스쿼트 20kg + 폼 점검 우선

**출처**
IPF Technical Rules Book 2024 (15kg 여성 바) ★★★

**회의 결정거리**
- 사용자가 "헬스장 여성 바 유무"를 어떻게 알지? 디폴트는 어느 쪽?
  - 옵션 A: 디폴트 20kg, "혹시 15kg 여성용 바 있으면 사용 가능" 메시지
  - 옵션 B: 디폴트 15kg, 없으면 사용자가 끌 수 있게
  - 옵션 C: 헬스장 등록 시 보유 장비 체크박스로 입력받기

## D14 — 첫 세션 무게 상한 클램프

**한 줄 요약**: 사용자가 첫 세션에서 입력한 무게가 종목별 상한값을 넘으면 자동으로 상한까지 떨군다. 평균 체중 완전 초보 남성이 첫 스쿼트 워크셋에서 65kg를 넘는 경우는 드물다는 Rippetoe의 가이드를 코드화.

**이게 뭐냐면**
- 첫 세션 욕심을 내면 폼이 무너지고 부상 위험. "들 수 있다"고 생각하는 무게보다 보수적으로 시작이 정석
- 종목별 캡 × (체중 / 기준체중) 식으로 체중 비례 보정

**언제 발동되나**
- 첫 세션 진입 시 항상
- 1RM 입력값이 캡을 초과해도 캡으로 떨굼

**필요한 입력**
- 성별, 체중, 훈련 이력

**처방 내용**
- 종목별 캡 × (체중 / 기준체중)
- 종목별 캡 (남/여, kg):
  - 스쿼트 65 / 35
  - 데드 70 / 40
  - 벤치 40 / 25
  - OHP 30 / 18
- 기준체중: 남 70kg, 여 55kg
- 사용자가 더 높게 입력해도 캡까지 떨굼 + "첫 세션 상한" 메모

**예시 시나리오**
- 남 80kg가 첫 스쿼트 100kg 요청 → 65 × (80/70) ≈ 74kg, "첫 세션 상한" 메모
- 여 60kg가 첫 데드 60kg 요청 → 40 × (60/55) ≈ 43.6kg로 클램프

**출처**
Mark Rippetoe, Starting Strength: Basic Barbell Training (Blue Book) ★★☆

**회의 결정거리**
- 캡이 너무 보수적인지(헬스 경력 0인 60kg 여성에게는 OK이지만 100kg 남성에게는?)
  - 옵션 A: 현재 캡 유지
  - 옵션 B: 100kg+ 사용자는 체중 비례 +20% 가산
  - 옵션 C: 캡 자체를 폐지하고 RPE로만 첫 세션 안전 가드

## D15 — 증량 폭 (초기)

**한 줄 요약**: 첫 세션 이후 다음 세션 무게는 하체 +5kg(초기 +10kg), 상체 +2.5kg(초기 +5kg)씩 올린다. 진입에서는 "초기 폭"만 사용한다.

**언제 발동되나**
- 첫 세션 처방 직후
- 매 세션 성공 후 (D15는 실시간 코칭에서도 사용 중)

**처방 내용**
- 하체(스쿼트·데드):
  - 초기 +10kg
  - 이후 +5kg
- 상체(벤치·OHP·로우):
  - 초기 +5kg
  - 이후 +2.5kg
- 실패 시 절반 또는 디로드
- 진입에선 "초기 폭"만 사용. 이후 폭은 미구현

**예시 시나리오**
- 60kg 스쿼트 5×5 성공 → 다음 세션 70kg (초기 +10kg)
- 60kg 벤치 5×5 성공 → 다음 세션 65kg (초기 +5kg)

**출처**
Mark Rippetoe, "Incremental Increases" ★★☆

**회의 결정거리**
- "이후 폭"으로 전환되는 시점(언제 +5 → +2.5kg로 떨어뜨릴지)
  - 옵션 A: 첫 stall(미달) 발생 시 자동 전환
  - 옵션 B: 4주 경과 시 자동 전환
  - 옵션 C: 사용자가 "이제 +2.5씩만 올려달라" 직접 입력

# 6단계 · 강도/볼륨

## C02 — 목표별 렙 범위 + %1RM

**한 줄 요약**: 목표를 받아 렙 범위와 %1RM을 자동 매핑한다. 스트렝스 1~6렙·85~100%, 벌크업/근육량 증가 6~12렙·67~85%, 다이어트 8~15렙·60~80%, 건강증진 12~20렙·50~67%.

**이게 뭐냐면**
- Repetition Continuum = "렙 범위와 강도 사이에 연속체가 있다"는 모델 (Schoenfeld 2021)
- 단, 근비대는 6~30렙 모두 효과적이라는 메타분석 결과도 있음 (N04 참고) — 너무 좁게 묶지 않음

**언제 발동되나**
- 진입 시 항상
- 세션 전

**필요한 입력**
- 사용자 목표

**처방 내용**
- 스트렝스 → 1~6렙 / 85~100%
- 벌크업·근육량 증가 → 6~12렙 / 67~85%
- 다이어트 → 8~15렙 / 60~80%
- 건강증진 → 12~20렙 / 50~67%

**예시 시나리오**
- 벌크업 사용자 → "6~12렙, %1RM 67~85" 처방
- 다이어트 사용자 → "8~15렙, %1RM 60~80" 처방

**출처**
Schoenfeld 2021 / NSCA ★★★

---

## C05 — 디폴트 RIR

**한 줄 요약**: 컴파운드 운동은 RIR 2(2개 더 가능), 아이솔레이션은 RIR 1로 기본 처방. 진입 함수는 항상 "compound"로 호출되어서 사실상 RIR 2 고정으로 떨어진다.

**이게 뭐냐면**
- RIR = Reps in Reserve, "남은 가능 횟수"
- 실패까지 가지 않아도 근비대 효과 거의 동일 (Refalo/Pelland 2024 메타) → 안전을 위해 RIR로 처방
- 컴파운드는 신경 부담↑ → RIR 2로 더 보수적
- 아이솔은 부담↓ → RIR 1까지 가능

**언제 발동되나**
- 진입 시 항상
- 세션 전

**처방 내용**
- 컴파운드 → RIR 2
- 아이솔레이션 → RIR 1
- 진입 함수는 항상 "compound"로 호출 → 사실상 RIR 2 고정

**출처**
Refalo / Pelland 2024 ★★★

**회의 결정거리**
- 운동마다 movement_type을 받아서 분기시켜야 함
  - 옵션 A: 운동 라이브러리에 movement_type 태그 추가하고 자동 분기
  - 옵션 B: 사용자가 운동마다 직접 선택
  - 옵션 C: 컴파운드/아이솔 자동 분류는 8단계 J05·J07 룰로 위임

## C06 — 근육군당 최소 빈도

**한 줄 요약**: 근육 한 부위는 최소 주 2회 자극해야 한다. 주 1회는 비효율이고, 빈도 자체보다 같은 볼륨을 분산했을 때 단백질 합성이 길게 유지된다는 게 근거.

**이게 뭐냐면**
- "주 2회"가 근비대·근력 연구에서 황금 기준으로 굳어진 빈도
- 같은 총 볼륨이라도 1회 몰빵보다 2~3회 분할이 단백질 합성을 길게 유지

**언제 발동되나**
- 진입 시 항상. 주간 스케줄 짤 때 가슴/등/다리 등 각 부위가 주 2회 이상 들어가는지 체크

**처방 내용**
- 주 2회 이상 (고정값, 사용자 레벨·목표 무관)

**예시 시나리오**
- 분할이 "월: 가슴 / 수: 등 / 금: 다리" → 가슴 주 1회만 → 룰 위반
- 시스템은 "월·목 가슴 / 화·금 등 / 수·토 다리" 같은 주 2회 분할로 재배치

**출처**
Schoenfeld 2016/2018 ★★★

---

## C07 — 휴식 시간 (목표별)

**한 줄 요약**: 목표가 스트렝스면 컴파운드 240초·아이솔 180초로 길게 쉬고, 그 외 목표는 컴파운드 150초·아이솔 90초로 더 짧게 쉰다. 스트렝스만 명확히 길다.

**이게 뭐냐면**
- 휴식 시간은 신경계 회복과 직결 — 1RM 가까운 무게는 더 오래 쉬어야 다음 세트 수행 가능
- Schoenfeld 2016: "1분 휴식보다 3분 휴식이 근비대·근력 모두 우월"

**언제 발동되나**
- 진입 시 항상
- 세션 전 (실시간 코칭에서도 휴식 타이머 자동 카운트다운으로 사용)

**필요한 입력**
- 운동 종류
- 목표

**처방 내용**
- 스트렝스:
  - 컴파운드 240초
  - 아이솔 180초
- 그 외 (벌크업·근육량 증가·다이어트·건강증진):
  - 컴파운드 150초
  - 아이솔 90초

**예시 시나리오**
- 다이어트 + 벤치프레스(컴파운드) → 휴식 150초
- 스트렝스 + 데드(컴파운드) → 휴식 240초
- 벌크업 + 바이셉컬(아이솔) → 휴식 90초

**출처**
Schoenfeld 2016 / de Salles ★★★

## B02 — 시작 볼륨 (근육군별 세트/주)

**한 줄 요약**: 새 메조사이클 시작 시 근육군별로 "성장이 시작되는 최소 볼륨(MEV)"을 처방한다. 가슴 8, 등 10, 어깨 8, 이두 8, 삼두 6, 다리 7, 둔근/햄 6, 복근 0.

**이게 뭐냐면**
- MEV = Minimum Effective Volume, 성장을 일으키는 최소 주간 세트 수
- 메조사이클 1주차는 MEV에서 시작 → 매주 1~2세트씩 추가 (B03 MAV 점진) → MRV(B04, 회복 가능 최대)에서 디로드
- 이 진입에서는 1주차 처방만 하므로 MEV만 호출

**언제 발동되나**
- 진입 시 항상
- 주간 단위 (메조 1주차)

**필요한 입력**
- 근육군

**처방 내용 (근육군당 시작 세트/주)**
- 가슴 8
- 등 10
- 어깨 8
- 이두 8
- 삼두 6
- 다리 7
- 둔근·햄 6
- 복근 0
- 기타 8
- 진입은 6대 근육군(가슴·등·어깨·다리·이두·삼두)에 일괄 적용

**출처**
Israetel ★★★

**회의 결정거리**
- 둔근·햄·복근·전완은 처방 안 됨, 어디 단계에서 넣을지
  - 옵션 A: 6대 근육군 처방 후 둔근/햄/복근/전완은 별도 가산 단계로 분리
  - 옵션 B: 모든 근육군을 일괄 처방으로 묶기
  - 옵션 C: 사용자 약점 입력에 따라 추가 근육군 활성화

## B05 — 권장 주간 볼륨 범위

**한 줄 요약**: 목표별로 근육군당 주간 세트 권장 범위가 다르다. 벌크업/근육량 증가 12~20세트, 스트렝스 8~16세트, 다이어트 10~18세트, 건강증진 8~12세트.

**이게 뭐냐면**

- 12~20세트가 권장 범위, 20+ 추가효과 미미

**언제 발동되나**
- 진입 시 항상
- 세션 전

**처방 내용**
- 벌크업·근육량 증가 → 12~20세트
- 스트렝스 → 8~16세트
- 다이어트 → 10~18세트
- 건강증진 → 8~12세트

**예시 시나리오**
- 스트렝스 사용자 → 근육군당 주 8~16세트 권장
- 벌크업 사용자 → 주 12~20세트

**출처**
Schoenfeld / Krieger ★★★

**예시 시나리오**

- 훈련 36개월 사용자 → 고급 → D03 Block

## D02 — DUP (중급)

**한 줄 요약**: 한 주 안에서 월·수·금 강도와 렙수를 다르게 돌려, 같은 부위를 다른 자극으로 3회 친다. 중급 정체기 깨는 데 효과적.

**이게 뭐냐면**
- DUP = Daily Undulating Periodization (일별 변화 주기화)
- 전통 주기화는 "이번 4주는 비대, 다음 4주는 근력"으로 블록을 길게
- DUP는 그걸 한 주 안에 압축 — 같은 운동을 월·수·금 다른 강도로
- 중급은 단일 자극에 빨리 적응 → 자극 다양화 효과 큼

**언제 발동되나**
- 7단계 주기화 모델 선택 시 사용자 레벨이 "중급"
- 주간 단위

**필요한 입력**
- 사용자 레벨(중·고급)

**한 주 구성 (벤치프레스 예시)**
- 월 — 비대(근비대 자극)
  - 8~12렙, %1RM 67~75%, 3~4세트
  - 목표: 근육량 증가
- 수 — 근력
  - 3~6렙, %1RM 85~92%, 3~5세트
  - 목표: 신경계 적응, 절대 근력
- 금 — 파워
  - 1~3렙, %1RM 90~100%, 3~5세트, 빠른 동심성
  - 목표: 폭발력·고중량 적응

**예시 시나리오**
- 입력: 훈련 14개월 사용자(중급), 벌크업, 주 4일, 1RM 벤치 90kg
- 시스템 처방:
  - 월: 벤치 60~67kg × 8~12렙 × 4세트
  - 수: 벤치 76~83kg × 3~6렙 × 4세트
  - 금: 벤치 81~90kg × 1~3렙 × 5세트

**왜 중급에만**
- 초보는 단순 선형(매 세션 +2.5kg)만 돌려도 계속 늚 — DUP는 변수만 늘림
- 고급은 한 주 안 변동으론 부족 — 더 긴 블록 주기(D03) 필요

**출처**
Painter 2012 / Colquhoun 2017 ★★★

**회의 결정거리**
- 월·수·금 고정 vs 사용자 가용 요일 회전?
  - 옵션 A: 주 3일 가용 → 고정 월수금
  - 옵션 B: 주 4일이면 추가 1일은 보조 운동만
  - 옵션 C: 주 2일이면 다른 모델로(비대·근력만 2일)

---

## D03 — Block (고급)

**한 줄 요약**: 3~4주 단위 블록(축적 → 변환 → 실현)으로 한 적응 자극에 집중한다. 고급 선수용. 한 블록에 1개 적응만 우선.

**이게 뭐냐면**
- Block Periodization = 적응 자극을 분리해 한 블록에 한 가지에만 집중하는 모델 (Issurin/Verkhoshansky)
- 고급은 한 주 변동(DUP)으로는 충분한 자극이 안 됨 → 더 긴 블록 단위 필요
- 축적기에 볼륨↑로 기초 쌓고, 변환기에 강도↑로 전환, 실현기에 최대 강도로 피크

**언제 발동되나**
- 7단계 주기화 모델 선택 시 사용자 레벨이 "고급"
- 주간 단위

**필요한 입력**
- 사용자 레벨(고급)

**처방 내용 (블록당 3~4주)**
- 축적기 (1~4주): 볼륨↑, 절대 강도↓
- 변환기 (5~7주): 강도↑, 볼륨 줄임
- 실현기 (8~10주): 최대 강도, 볼륨 최저

**예시 시나리오**
- 훈련 36개월 사용자, 대회 준비 → 10주 블록 사이클
  - 1~4주차: 12렙·70% × 5세트 (볼륨↑)
  - 5~7주차: 5렙·85% × 4세트 (강도↑)
  - 8~10주차: 1~3렙·95% × 3세트 (피크)

**출처**
Issurin / Verkhoshansky ★★☆

## G01 — 디로드 정의

**한 줄 요약**: 디로드 = 의도적 훈련 자극 감소 (5~7일). 평균 4~6주마다 시행. 사용자 누적 피로를 빼고 다음 메조에 더 큰 자극을 받기 위한 휴지기.

**언제 발동되나**
- 진입 시 항상
- 주간 단위로 디로드 주차 스케줄링

**필요한 입력**
- 주차 누적
- 사용자 입력

**처방 내용**
- 디로드 기간: 5~7일
- 주기: 4~6주마다 (메조사이클당 1회)

**출처**
Bell et al / Frontiers ★★★

## G10 — 한국 사용자 디로드 주기

**한 줄 요약**: 절대초보·초보는 정해진 주기 없이 신호 기반 자동(G07), 중·고급은 4~6주마다 강제 디로드. 한국 직장인 누적 피로·회식·출장을 고려해 6주 상한으로 묶음. 진입에선 "5주 훈련 + 1주 디로드 = 메조 6주"로 박힘.

**이게 뭐냐면**
- 초보는 신경 적응 단계라 강제 디로드보다 신호(피로·통증) 기반이 유리
- 중급+ 한국 직장인은 회식·출장 등 외부 변수가 많아 6주 상한이 안전

**언제 발동되나**
- 메조 설계 시 1회

**필요한 입력**
- 사용자 레벨
- 메조 시작 주차

**처방 내용**
- 초보 → 신호 기반 자동 (정해진 주기 없음)
- 중급 이상 → 강제, 6주마다
- 진입에선 "5주 훈련 + 1주 디로드 = 메조 6주"로 박힘
- 5주차 종료 후 "다음 주 디로드 추천" 푸시 + G05 처방 미리 채움

**예시 시나리오**
- 직장인 중급자 메조 시작 → 5주 훈련 + 1주 디로드 = 6주 메조 자동 설정

**출처**
Renaissance Periodization 메조 가이드 ★★☆

**회의 결정거리**
- 초보의 "신호 기반 자동" 디로드는 어떤 신호로 발동할지(피로 점수·수면·통증 어떤 입력으로?)
  - 옵션 A: G07 다지표(RPE drift·수면·통증·동기·PR 경과)로 자동
  - 옵션 B: 단일 지표(통증 NRS) 임계 초과만 보고 발동
  - 옵션 C: 초보도 강제 6주 주기로 통일

# 8단계 · 운동 선택

## J05 — 컴파운드:아이솔 비율

**한 줄 요약**: 한 세션에 들어갈 컴파운드와 아이솔 운동 비율을 레벨별로 정한다. 초보 70:30, 중급 60:40, 고급 50:50. 초보일수록 컴파운드(다관절·고효율) 비중이 높다.

**이게 뭐냐면**
- 컴파운드 = 다관절 운동(스쿼트·벤치·데드·OHP). 신경계 적응·전신 자극 효과↑
- 아이솔레이션 = 단관절 운동(바이셉 컬·트라이셉 익스텐션). 약점 부위 보강
- 초보는 컴파운드만으로 충분, 고급은 약점 보완을 위해 아이솔 비중↑

**언제 발동되나**
- 진입 시 항상
- 세션 전

**필요한 입력**
- 사용자 레벨
- 약점 부위

**처방 내용**
- 초보 → 70:30
- 중급 → 60:40
- 고급 → 50:50

**예시 시나리오**
- 중급 사용자 + 5종목 처방 → 컴파운드 3 / 아이솔 2
- 초보 + 5종목 → 컴파운드 4 / 아이솔 1 (반올림)
- 고급 + 6종목 → 컴파운드 3 / 아이솔 3

**출처**
Israetel / NSCA ★★☆

## J03 — 자유중량:머신 비율

**한 줄 요약**: 자유중량과 머신을 어떤 비율로 섞을지 결정한다. 초보 또는 당뇨·고혈압 환자는 안전 위해 머신↑(40:60), 스트렝스는 자유중량↑(80:20), 그 외 60:40.

**이게 뭐냐면**
- 자유중량 = 바벨·덤벨로 안정성을 사용자가 컨트롤. 근력 전이 우월
- 머신 = 가이드 레일 있어 안전성↑, 단독훈련 가능
- 근비대 효과는 둘이 동등 (Schwanbeck 2020 메타)

**언제 발동되나**
- 진입 시 항상
- 세션 전

**필요한 입력**
- 사용자 목표/환경/경험

**처방 내용**
- 초보 또는 당뇨·고혈압 → 40:60 (머신 비중↑)
- 스트렝스 → 80:20
- 그 외 → 60:40

**예시 시나리오**
- 초보 + 건강증진 → 40:60
- 중급 + 스트렝스 → 80:20
- 중급 + 벌크업 → 60:40
- (안전 우선) 중급 + 고혈압 → 40:60 (질환이 레벨/목표보다 우선)

**출처**
Schwanbeck 2020 / Haugen 2023 ★★★

**회의 결정거리**
- 초보 vs 질환 둘 다 해당하면 같은 40:60인데, 사용자 안전 측면에서 더 보수적 분기 필요?
  - 옵션 A: 초보 + 질환은 30:70까지 머신 비중↑
  - 옵션 B: 현재 그대로 40:60
  - 옵션 C: 질환별 차등 (당뇨 50:50, 고혈압 30:70)

- 바벨 있음 → 원래 운동 유지
- 바벨 없음 → 덤벨 → 케틀벨 → 밴드 → 맨몸 순서로 대체

**예시 시나리오**

## I03 — 운동 순서 (컴파운드 우선)

**한 줄 요약**: 한 세션 안에서 운동을 컴파운드 → 보조 컴파운드 → 머신 컴파운드 → 아이솔 순서로 자동 정렬한다. 신경피로·안전성 이유.

**언제 발동되나**

- 세션 전

**필요한 입력**
- 세션 운동 목록

**처방 내용 (가중치 표로 정렬)**
- 빅3 = 1
- 보조 컴파운드 = 2
- 머신 컴파운드 = 4

**예시 시나리오**

# 9단계 · 워밍업/스트레칭

## I05 — 일반 워밍업

**한 줄 요약**: 본 운동 전 5~10분 저강도 유산소(조깅·자전거)로 심박·근온도·관절액을 가동한다. 워밍업 시간은 세션 가용분의 1/6 정도로 자동 산정.

**언제 발동되나**
- 운동 시작 직전 항상
- 진입 시 항상

**필요한 입력**
- 사용자 첫 메인 운동
- 가용 시간

**처방 내용**
- 워밍업 시간 = `min(10, max(5, 세션가용분/6))` 분
- 형태: 저강도 유산소 (조깅·자전거 등)
- 땀 살짝 날 정도

**예시 시나리오**
- 세션 60분 → max(5, 10) = 10분 워밍업
- 세션 30분 → max(5, 5) = 5분 워밍업
- 세션 90분 → min(10, 15) = 10분 워밍업 (상한 10분 클램프)

**출처**
NSCA, Introduction to Dynamic Warm-Up ★★★

## I06 — 컴파운드 워크업

**한 줄 요약**: 메인 컴파운드 첫 세트 직전, 빈 봉부터 점진 증량으로 워크업 세트를 깐다. 빈 봉 5렙×2세트 → 40%×5렙 → 60%×3렙 → 80%×2렙 → 본 세트.

**이게 뭐냐면**
- 워크업 = 본 세트 무게에 신경계·관절을 적응시키는 과정. 부상 예방·수행 향상 모두 효과
- Starting Strength 표준 워크업 시퀀스

**언제 발동되나**
- 시작 무게에 "스쿼트" 있을 때 (현재 진입은 스쿼트 한정)
- 메인 컴파운드 첫 세트 직전

**필요한 입력**
- 본 세트 무게(kg)
- 종목(스쿼트/벤치/데드/프레스)

**처방 내용 (스쿼트 무게 기준)**
- 빈 봉 5렙 × 2세트
- 40% × 5렙
- 60% × 3렙
- 80% × 2렙
- 본 세트

**예시 시나리오**
- 본 세트 100kg 스쿼트 → 빈 봉 5×2 → 40kg×5 → 60kg×3 → 80kg×2 → 100kg

**출처**
Mark Rippetoe, Starting Strength — Warmup ★★☆

## P01 — 워밍업 스트레칭 정책

**한 줄 요약**: 워밍업에는 동적 스트레칭만, 정적 스트레칭은 운동 후로 배치한다. 운동 전 정적 스트레칭(60초+)은 근력을 떨어뜨린다는 메타분석이 근거.

**언제 발동되나**
- 진입 시 항상
- 세션 전

**필요한 입력**
- 세션 시점

**처방 내용**
- 워밍업 정적 스트레칭 회피
- 워밍업 동적 스트레칭 권장
- 정적 스트레칭은 운동 후로 배치

**출처**
Behm 2016 메타분석 ★★★

## P05 — 정적 스트레칭 경고 (메시지)

**한 줄 요약**: 본 세트 60분 이내에 정적 스트레칭 45초 이상 하면 최대 근력이 평균 -5.4% 떨어진다는 메시지를 띄운다.

**언제 발동되나**
- 진입 시 항상 (메시지로 표시)
- 본 세트 직전 60분 이내

**처방 내용**
- 메시지 표시: "워밍업 정적 스트레칭 ≥ 45초 회피 — 최대 근력 평균 -5.4%"

**출처**
Simic et al. 2013, Scand J Med Sci Sports — meta-analysis ★★★

# 10단계 · 카디오

## O01 — 간섭 검사

**한 줄 요약**: 카디오와 근력의 "간섭효과"를 검사한다. 다이어트·건강증진은 주 2회 30분, 그 외(벌크업·근육량 증가·스트렝스)는 주 1회 30분으로 처방. 주 < 3회 또는 ≤ 30분이면 간섭 미미.

**이게 뭐냐면**
- 간섭효과 = 카디오가 근력 적응을 방해하는 현상 (Wilson 2012 메타)
- 주 3회 미만/30분 이하 카디오는 간섭 미미
- 같은 날엔 근력 먼저 또는 6시간 이상 분리

**언제 발동되나**
- 진입 시 항상
- 주간

**필요한 입력**
- 사용자 카디오 빈도/시간

**처방 내용**
- 다이어트·건강증진 → 카디오 주 2회
- 그 외 (벌크업·근육량 증가·스트렝스) → 주 1회
- 카디오 1회당 분 = 30 (고정)
- 안전 판정: 주 < 3회 또는 ≤ 30분 → 안전
- 메시지: "근력 먼저 또는 6시간↑ 분리"

**예시 시나리오**
- 다이어트 사용자 → 주 2회 × 30분 카디오 + "근력 후 또는 6시간↑ 분리"
- 벌크업 사용자 → 주 1회 × 30분
- 스트렝스 사용자 → 주 1회 × 30분

**출처**
Wilson 2012 메타분석 ★★★

## O02 — HIIT vs LISS 선택

**한 줄 요약**: 카디오 가용분과 근력 빈도에 따라 HIIT(고강도 인터벌)와 LISS(저강도 지속)를 자동 분기. 시간 짧으면 HIIT, 근력 빈도 높으면 회복 부담 적은 LISS.

**이게 뭐냐면**
- HIIT = 고강도 인터벌. 시간 효율 높지만 회복 부담↑
- LISS = 저강도 지속(걷기·자전거). 회복 쉬움, 시간 길음
- 근력 사용자는 회복 여력 위해 LISS 우선 권장

**언제 발동되나**
- 진입 시 항상
- 주간

**필요한 입력**
- 사용자 가용 시간
- 회복 상태

**처방 내용**
- 카디오 가용분 < 20 → "HIIT 15분"
- 근력 빈도 ≥ 4 → "LISS 30분"
- 그 외 → "HIIT 1~2회 + LISS 1~2회 혼합"

**예시 시나리오**
- 주 5일 근력 사용자 → "LISS 30분"
- 시간 부족(가용 15분) → "HIIT 15분"
- 주 3일 근력 + 가용 30분 → "HIIT 1~2회 + LISS 1~2회 혼합"

**출처**
Schoenfeld / Helms ★★★

## O03 — 카디오 타이밍

**한 줄 요약**: 카디오는 근력 후 또는 별도 세션. 같은 부위 카디오는 회피. 진입에선 strength_focus=True, leg_day=False 고정.

**언제 발동되나**
- 진입 시 항상
- 세션 전후

**필요한 입력**
- 세션 구성

**처방 내용**
- 입력 고정: strength_focus = True, leg_day = False
- 메시지: "근력 후 카디오 또는 별도 세션"

**출처**
Robineau 2016 ★★☆

**회의 결정거리**

  - 옵션 A: 매 세션 leg_day 자동 감지(처방된 운동 보고)
  - 옵션 B: 사용자가 "오늘 다리" 선택 시만 회피 메시지
  - 옵션 C: 현재 그대로 고정 메시지

**한 줄 요약**: 성인은 7~9시간 수면 권장. 6시간 미만이 4주 지속되면 근감소·테스토스테론↓·인슐린저항↑.

- 일일

**필요한 입력**

**처방 내용**
- 권장 수면 7~9시간 (나이 무관 고정)

**예시 시나리오**

Knowles 2018 / Mah 2011 ★★★

## K07 — 능동 회복

**한 줄 요약**: 회복일에 저강도 유산소 10~30분(VO2max의 20~40%)을 권장한다. 완전 휴식보다 회복 속도가 빠르다는 메타분석이 근거.

**이게 뭐냐면**
- 능동 회복 = 가벼운 활동으로 혈류↑·노폐물 제거 → 회복 가속
- 완전 휴식보다 가벼운 자전거·걷기·수영이 다음 세션 수행에 유리

**언제 발동되나**
- 진입 시 항상
- 세션 후 / 회복일

**필요한 입력**
- 사용자 피로도

**처방 내용**
- 강도: 20~40% VO2max
- 시간: 10~30분
- 활동: 가벼운 자전거·걷기·수영

**예시 시나리오**
- 회복일에 가벼운 자전거 20분 또는 걷기 30분 추천

**출처**
Dupuy 2018 / Frontiers ★★★

## Q04 — 요추 통증 프로토콜

**한 줄 요약**: 허리 통증 사용자를 발생 일수에 따라 급성/아급성/복귀 3단계로 나누고, 단계별로 허용·회피 동작과 재시작 무게(%1RM)를 다르게 처방한다. 적색기(다리 양측 저림·안장마비·괄약근 이상 등)는 차단·응급실.

**이게 뭐냐면**
- NICE NG59 핵심 권고: 침상안정 회피, 활동 유지가 1차. 운동요법은 1차 권고
- McKenzie MDT = 통증 중심화(centralisation)를 유도하는 방향성 운동 (LBP 환자 70%+가 신전 선호)
- 마미증후군(cauda equina) = 응급. 안장마비·양측 다리 저림·괄약근 이상이 핵심 증상
- 적색기 5종: 다리 양측 저림 / 안장마비 / 괄약근/배뇨 이상 / 발열·감염 의심 / 외상 후·암 병력 동반

**언제 발동되나**
- 부상 이력에 "허리" 체크 시
- 통증 발생 시 / 재활 단계별

**필요한 입력**
- 통증 부위·강도(NPRS 0~10)·발생 동작·일수·다리 저림 유무

**처방 내용 (단계별)**
- 적색기 신호 → 차단, 응급실/신경외과
  - 다리 양측 저림
  - 안장마비
  - 괄약근 이상
  - 발열
  - 암 병력 등
- 급성기 < 3일 → 일상 활동 + 골반 틸트 / 데드·스쿼트 회피
  - McKenzie 엎드려 신전 5×10회 시도
- 아급성 3~14일 → 스트레칭·근력·유산소·운동조절
  - 신경역동 운동·도수치료는 운동의 보조로만
- 복귀 ≥ 14일 → 글루트·코어 활성화, 데드·스쿼트는 1RM 30%부터, 주 +10% 증가

**예시 시나리오**
- 입력: 허리 부상 30일 경과 + 적색기 없음 + NPRS 2
  - 시스템 출력: 글루트·코어 활성화 + 1RM 30%부터 데드·스쿼트, 주 +10% 점진
- (다른 케이스) 입력: 데드 110kg 후 허리 통증 NPRS 6, 어제 발생
  - 시스템 출력: "급성기. 오늘은 가벼운 보행·골반 틸트·McKenzie 엎드려 신전 5×10회 시도. 데드/스쿼트 일시 중지. 3일 후 통증 NPRS 3 이하면 30kg(추정 1RM 30%)부터 RDL 재개"
- (적색기) 다리 저림이 종아리 아래로 내려가거나 양측 → 차단·의료기관 상담

**출처**
NICE NG59 (2016/2020 surveillance) + Foster 2018 Lancet LBP series ★★★

## Q05 — 어깨 RCRSP 프로토콜

**한 줄 요약**: 어깨 임핀지·회전근개 통증 사용자를 발생 일수에 따라 급성(<7일)/아급성(7~42일)/복귀(≥42일) 3단계로 처방한다. 단계적 운동 부하가 수술과 동등 효과(Lewis 2016).

**이게 뭐냐면**
- RCRSP = Rotator Cuff Related Shoulder Pain. 임핀지 증후군·회전근개 건병증·부분 파열을 통합한 진단명 (Lewis 2016)
- 핵심 위험인자 3개: GIRD(글레노휴메럴 내회전 결손), 외회전근 약화, 견갑운동이상 (Cools 2015)
- 권고 운동: 후방관절낭 스트레칭(sleeper/cross-body), 외회전 강화(밴드 ER), 견갑 안정화(서라투스 펀치·로우)
- 적색기: 외상 후 거상 불가·진행성 위약·야간통+종괴·발열·신경학적 결손

**언제 발동되나**
- 부상 이력에 "어깨" 체크 시
- 통증 발생 시 / 재활 단계별

**필요한 입력**
- 통증 부위·강도(NPRS)·유발 동작·외상 유무

**처방 내용 (단계별)**
- 적색기 신호 → 차단, 정형외과
  - 외상 후 거상 불가
  - 진행성 위약
  - 야간통 + 종괴
  - 발열
  - 상지 신경학적 결손
- 급성 < 7일 → 오버헤드 회피, 진자운동
  - behind-the-neck·딥 호리젠탈 어덕션 회피
  - 통증 허용 ROM 내 진자운동·견갑 후퇴/하강 활성
- 아급성 7~42일 → 밴드 외회전·서라투스 펀치·로우, NPRS 상한 4
  - GIRD 교정 (cross-body·sleeper stretch)
  - 견갑 안정화 (서라투스 펀치)
- 복귀 ≥ 42일 → 벤치·OHP·풀업 1RM 30%부터, 주 +10%
  - 오버헤드 복귀는 통증 0/10 + 외회전 좌우차 < 10% 확인 후

**예시 시나리오**
- 입력: 어깨 부상 14일 경과 + 적색기 없음 + NPRS 3
  - 시스템 출력: 밴드 외회전 3×15·서라투스 펀치·로우 + NPRS 상한 4
- (다른 케이스) 입력: 오버헤드 프레스 50kg 셋업 시 어깨 앞쪽 찌릿 NPRS 5
  - 시스템 출력: "급성기. 오버헤드 1주 중지, behind-the-neck·업라이트 로우도 회피. 대신 풀업 행잉·진자운동·밴드 외회전 3×15. 1주 후 통증 감소 시 덤벨 OHP를 1RM 30%부터 재시작"
- (적색기) 야간통 심해지거나 팔 들 수 없으면 차단·정형외과 의뢰

**출처**
Lewis 2016 Man Ther + Cools 2015 Braz J Phys Ther ★★★

## Q06 — 무릎 PFPS 프로토콜

**한 줄 요약**: 슬개대퇴 통증(PFPS) 사용자를 발생 일수에 따라 급성(<2주)/아급성(2~6주)/복귀(≥6주) 3단계로 처방한다. 고관절+슬관절 복합 운동이 단독 슬관절 운동보다 우월(Crossley 2016).

**이게 뭐냐면**
- PFPS = Patellofemoral Pain Syndrome. 슬개대퇴 관절 응력 증가에 의한 무릎 앞쪽 통증
- Powers 2017 BJSM 합의: 응력 증가 인자는 근위(고관절)·국소(슬관절)·원위(족부) 3가지
- 핵심 운동: 클램쉘·사이드 라잉 힙 어브덕션·몬스터 워크(고관절 외전·외회전) + 미니 스쿼트 0~45°(대퇴사두)
- 적색기: 외상 후 부종·잠김(locking)·이탈감·체중부하 불가·발적/발열

**언제 발동되나**
- 부상 이력에 "무릎" 체크 시
- 통증 발생 시 / 재활 단계별

**필요한 입력**
- 통증 부위·강도(NPRS)·유발 동작·부종/잠김 유무

**처방 내용 (단계별)**
- 적색기 신호 → 차단, 정형외과
  - 외상 후 부종
  - 관절잠김
  - 체중부하 불가
- 급성 < 2주 → 통증 유발 동작 회피
  - 딥 스쿼트·런지·계단 내려오기·오픈체인 레그익스텐션 90°→0° 회피
  - 활동 수정·교육 우선
- 아급성 2~6주 → 클램쉘·힙 어브덕션·미니 스쿼트 0~45°
  - 고관절(외전·외회전) + 대퇴사두(짧은 ROM) 복합 강화
  - 통증 NPRS ≤ 3 허용
  - 단기 진통은 슬개골 테이핑 보조 가능
- 복귀 ≥ 6주 → 스쿼트·런지 ROM 점진 확장
  - 무게는 통증 무관 자세 가능 시 1RM 30%부터 주 +10%
  - 풀스쿼트는 통증 0/10 + 편측 근력차 < 10% 확인 후

**예시 시나리오**
- 입력: 무릎 부상 30일 경과 + 적색기 없음 + NPRS 2
  - 시스템 출력: 클램쉘·힙 어브덕션·미니 스쿼트 0~45° 처방
- (다른 케이스) 입력: 백스쿼트 80kg 바닥에서 무릎 앞쪽 통증 NPRS 5, 계단 내려올 때도 아픔
  - 시스템 출력: "아급성기 진입. 풀ROM 스쿼트·계단 내려오기 2주 회피. 클램쉘 3×15·사이드 라잉 힙 어브덕션 3×15·미니 스쿼트(0~45°) 3×10·레그프레스 짧은 ROM 24kg(1RM 30%)부터. 4주 후 통증 NPRS≤2면 박스 스쿼트(평행)부터 점진 ROM 확장"
- (적색기) 무릎 잠김/부종 새로 생기면 차단·정형외과 의뢰

**출처**
Powers 2017 BJSM consensus + Crossley 2016 BJSM Part 2 + Barton 2015 BJSM ★★★

## R10 — 한국 직장인 회식 대응

**한 줄 요약**: 한국 직장인 평균 근로 38~40시간, 회식 월 2~4회를 전제로 회식 당일은 운동 패스, 익일은 20분 압축 루틴, 주 볼륨 -10~15% 자동 보강.

**이게 뭐냐면**
- KOSIS 임금근로일자리 통계: 주당 평균 근로시간 약 38~40시간
- Parr 2014: 알코올 24시간 내 근비대 신호(MPS) -37% 감소
- 진입에선 "오늘 회식 = False" 고정 호출 → 사실상 정상 진행 메시지만 출력

**언제 발동되나**
- 진입 시 항상 (현재는 회식=False 고정)
- 주간 일정 조정 시

**필요한 입력**
- 사용자의 근무 형태
- 회식 일정(캘린더 연동)
- 주당 운동 가능 일수

**처방 내용**
- 회식 = False (현재 진입에서 고정) → "오늘 정상 진행, 주간 볼륨 유지"
- 회식 = True (가정) →
  - 당일 휴식 권고
  - 익일 20분 압축 루틴(메인 운동 3개 5x5만)
  - 주 볼륨 -10~15% 자동 조정으로 미달 보강

**예시 시나리오**
- 사용자 '오늘 회식이라 운동 못 가요' → 챗봇 '내일 20분 압축 세션(스쿼트·벤치·로우 5x5)으로 주간 볼륨 92% 회복 가능. 음주 후 24시간은 근비대 신호가 줄어드니 폼 위주로'

**출처**
KOSIS 임금근로일자리 근로시간 통계 + 자체 룰 (Parr 2014 알코올·근단백질합성 참고) ★☆☆

**회의 결정거리**
- 사용자 회식 일정을 어떻게 입력받지? 매일 묻기? 캘린더 연동?
  - 옵션 A: 매일 "오늘 회식 있나요?" 푸시
  - 옵션 B: 구글/네이버 캘린더 연동해서 "회식" 키워드 자동 감지
  - 옵션 C: 사용자가 회식 직후 "회식 있었음" 토글로 보고

## N02 — 신경 적응 vs 근비대 기대값 메시지

**한 줄 요약**: 훈련 개월에 따라 사용자에게 적응 단계 메시지를 다르게 띄운다. 4~8주는 신경 적응(사이즈 변화 작음), 8주+ 근비대 시작, 8개월+ 누적은 크지만 진보 느림.

**이게 뭐냐면**
- 신경 적응 = 같은 무게를 더 잘 들 수 있게 신경계가 동원하는 단계. 사이즈 변화는 작음
- 근비대 적응 = 근육 단면적 자체가 늘어나는 단계. 보통 8주 이후
- 사용자 기대 관리에 핵심 — 초기 4주 사이즈 안 늘어도 정상이라는 안내

**언제 발동되나**
- 진입 시 항상

**필요한 입력**
- 훈련 기간

**처방 내용 (메시지 분기)**
- 훈련 < 2개월 → "초기 4~8주는 신경 적응, 사이즈 변화 작음"
- 훈련 2~8개월 → "8주부터 본격적 근비대"
- 훈련 ≥ 8개월 → "고급 단계, 누적은 크지만 진보 느림"

**예시 시나리오**
- 훈련 4주차 사용자 → "초기 4~8주는 신경 적응, 사이즈 변화 작음" 메시지
- 훈련 6개월 → "8주부터 본격적 근비대" 메시지
- 훈련 18개월 → "고급 단계, 누적은 크지만 진보 느림" 메시지

**출처**
Sale 1988 / Counts 2017 ★★★

## N04 — 근섬유 타입 교육

**한 줄 요약**: 근비대는 6~30렙 모두 효과적이며, "저렙=속근만 자극" 단순화는 오류라는 메시지를 띄운다.

**이게 뭐냐면**
- Type II(속근) 섬유도 고렙(15+)에서 비대
- Type I(지근) 섬유도 모든 렙 범위에서 비대
- "저렙=속근, 고렙=지근" 이분법은 단순화 오류 (Schoenfeld 2021)

**언제 발동되나**
- 진입 시 항상 (지식베이스 메시지)

**처방 내용**
- 메시지 표시: "근비대는 6~30렙 모두 효과적. 저렙=속근만 자극은 단순화 오류"

## M03 — 1RM 측정 방식

**한 줄 요약**: 사용자 레벨/나이에 따라 1RM 측정 방식을 분기. 초보·60세 이상은 AMRAP 추정만, 중급은 8주마다 AMRAP, 고급은 직접 1RM 테스트 옵션.

**이게 뭐냐면**
- 직접 1RM = 가장 정확하지만 부상 위험
- AMRAP = As Many Reps As Possible. 정해진 무게로 한계까지 반복 → Epley/Brzycki 공식으로 1RM 추정 (±5% 정확)
- 초보·노인은 부상 위험으로 직접 측정 회피

**언제 발동되나**
- 진입 시 항상
- 초기 셋업 / 정기 평가

**필요한 입력**
- 사용자 레벨
- 위험 허용도

**처방 내용**
- 초보 또는 60세 이상 → "AMRAP로 추정만"
- 중급 → "AMRAP 8주마다"
- 고급 → "1RM 직접 테스트 옵션"

**예시 시나리오**
- 65세 중급 사용자 → 60세 컷오프 우선 → "AMRAP로 추정만"
- 28세 중급 → "AMRAP 8주마다"
- 30세 고급 → "1RM 직접 테스트 옵션"

**출처**
Reynolds 2006 / NSCA ★★★

**회의 결정거리**
- "초보·60세↑" vs "중급·고급"이 겹칠 때(예: 65세 고급) 어느 쪽이 우선인지
  - 옵션 A: 나이 우선 (65세 고급 → AMRAP만)
  - 옵션 B: 레벨 우선 (65세 고급 → 직접 테스트 옵션)
  - 옵션 C: 둘 다 만족시키는 보수안 (65세 고급 → AMRAP 8주마다)

## 강도 처방 영역 (C01·C03·C04·C08·C09·C10·C11)

### C01 — NSCA RM-1RM% 표
- 무엇: 5RM=87% / 8RM=80% / 10RM=75% / 12RM=67% (선형 보간)
- 왜 안 부름: 추천은 렙범위·%1RM 범위만 출력하고 구체 무게는 안 처방. 사용자가 1RM 가지면 ×0.8 박는 단순 식만 씀
- 결정거리: 렙별 정확한 무게(예: "8렙 RPE 8 → 80kg")까지 처방할지

### C03 — NSCA 단계별 처방 (근력기·근비대기)
- 무엇: 근력기 80~95%·2~6렙·2~6세트 / 근비대기 50~75%·8~20렙·3~6세트
- 왜 안 부름: 진입에서 사용자에게 "지금 어떤 단계?"를 묻지 않음. 주기화 모델(D02·D03)에 단계가 자동 박혀 있어 중복
- 결정거리: 사용자가 단계 직접 선택할 옵션 줄지

### C04 — ACSM 일반인 가이드
- 무엇: 건강 성인 디폴트 (주 2~3회·근육군당 1~4세트·8~20렙·40~70% 1RM)
- 왜 안 부름: "건강증진" 목표를 C02·B05가 이미 처방하고 있어 중복. C04는 더 보수적
- 결정거리: 건강증진 사용자에게 C02 vs C04 중 어느 쪽 디폴트로

### C08 — Double Progression
- 무엇: 모든 세트가 렙 상한 도달 → 무게 +2.5%, 렙 하한 리셋
- 왜 안 부름: 한 세션 끝나고 다음 세션 처방 시점 룰. 추천은 1주차 처방만
- 어디서 호출돼야: 매 세션 종료 후 다음 세션 처방 진입

### C09 / C10 — 1RM 추정 공식 (Epley / Brzycki)
- 무엇:
  - C09 Epley: 1RM = 무게 × (1 + 렙/30). 6~10렙에서 정확
  - C10 Brzycki: 1RM = 무게 × 36/(37-렙). 1~6렙에서 정확
- 왜 안 부름: 추천 진입은 사용자가 1RM을 입력했다고 가정. 안 입력하면 D11/D12 빈 바벨로 떨어짐
- 결정거리: AMRAP 결과(예: "60kg × 8렙 가능") 받아 자동 1RM 계산해줄지 (M03가 "AMRAP 추정만" 권고하는 사용자 그룹에 필수)

### C11 — 세션 길이로 운동 가짓수 결정
- 무엇: 30~40분 → 4종목 / 45~60분 → 5종목 / 60~75분 → 7종목 / 75분+ → 8종목
- 왜 안 부름: 추천은 세션당 운동 가짓수 자체를 출력하지 않음. 프로그램(D06 등)이 박힌 가짓수 그대로
- 결정거리: 사용자 가용시간에 맞춰 운동 수 줄여줄지

- 무엇: 렙 하한 2세션 연속 미달 → -7.5%
- 왜 안 부름: 매 세션 후 처방 단계 룰

> **공통**: C08·C12·C13·C14·C15는 모두 "다음 세션 처방" 진입에 묶여야 함. 현재 그 진입 자체가 없음

## 주기화 모델 (D01·D03·D04·D09·D13)

### D01 — Linear Periodization (12주 사이클)
- 무엇: 1~4주 비대(8~12렙·67~75%) → 5~8주 근력(3~6렙·85~92%) → 9~12주 파워(1~3렙·90~100%)
- 왜 안 부름: 진입 7단계는 레벨로만 모델 분기 (초보 LP / 중급 DUP / 고급 Block). 사용자가 "12주 LP 사이클" 선택할 옵션 없음
- 결정거리: D01을 어느 사용자에게 띄울지

### D03 — Block Periodization (이미 호출됨)
- 진입 7단계에서 고급 사용자에게 호출 중. 정정: 이건 호출됨

### D04 — Flexible DUP
- 무엇: 사용자가 "오늘 컨디션 좋음/보통/피곤" 입력 → 그날 세션 타입 (근력/비대/회복) 동적 분기
- 왜 안 부름: 컨디션 입력은 매 세션 시작 시점에 받아야. 추천은 1회 호출이라 매일 묻지 못 함
- 어디서 호출돼야: 매 세션 시작 진입

### D09 — nSuns LP
- 무엇: 5/3/1 정체기 돌파용. 주 6일·메인+보조 모두 5/3/1 패턴
- 왜 안 부름: 진입 4단계 분기에 D09가 없음. 5/3/1(D07) → GZCLP(D08) → PHUL/PHAT(D10) 만 있음
- 결정거리: D07 stall 시 자동으로 D09로 전환할지

### D13 — 빈 바벨 워크업
- 무엇: 빈 바 → 50% → 70% → 90% → 100% 워크업 시퀀스
- 왜 안 부름: 추천 9단계는 I06(컴파운드 워크업)을 쓰는데, I06이 더 자세한 워크업(빈봉×5×2 + 40/60/80%)이라 D13 중복
- 결정거리: D13은 삭제해도 되는 중복 룰일 수 있음

## 운동 선택 보조 (F05·J04·J07·E14·E17)

### F05 — SFR (자극/피로 비율)
- 무엇: 운동별 SFR 점수 (1~5). 컴파운드 SFR 보통, 머신/아이솔 SFR 우수
- 왜 안 부름: 진입 8단계에서 운동 풀 선택 시 SFR 가중치를 안 씀. J05(컴파운드:아이솔 비율)만 사용
- 결정거리: 부상·피로 누적 사용자에게 SFR 높은 운동 우선 추천할지

### J04 — 편측 vs 양측 운동
- 무엇: 좌우 불균형 ≥ 10% → 편측 비중 ↑ (양측 50:편측 50). 그 외 75:25
- 왜 안 부름: `좌우_불균형_pct` 입력 자체가 진입에서 안 쓰임
- 결정거리: 좌우 불균형을 측정해서 받을지(앱에서 어떻게?)

### J07 — 근육군 매핑 (운동 → primary/secondary)
- 무엇: 백스쿼트=다리·둔근(primary), 복근·햄(secondary). 가중치 1.0/0.5
- 왜 안 부름: 진입 6단계 시작 볼륨(B02)을 근육군 단위로만 처방. 운동→근육 매핑은 안 씀
- 어디서 호출돼야: 누적 볼륨 추적 시 (운동 1세트 → 근육군별 가중치 분배)

### E14 — 가동범위(ROM) 디폴트
- 무엇: "풀ROM 우선, 신장강조 부분반복은 옵션" 메시지
- 왜 안 부름: 진입에서 ROM 안내 문구를 안 띄움
- 결정거리: 사용자에게 ROM 가이드 메시지 노출할지

### E17 — 그립 너비 가이드
- 무엇: 운동별 권장 그립 (벤치 어깨너비 1.5배 / 데드 어깨너비 후크 / 풀업 광배 와이드 / 풀업 이두 내로우)
- 왜 안 부름: 진입에서 그립 안내를 안 띄움 (실시간 코칭에선 이미 사용 중)
- 결정거리: 추천에서도 운동마다 그립 1줄 띄울지

## 디로드 트리거 다양화 (G03·G04·G06·G08·G09·G11)

### G03 — 디로드 처방 방법 (볼륨 -50%, 강도 -10~20%)
- 무엇: G05보다 더 일반적인 처방
- 왜 안 부름: 진입은 G05(세트 ½, 무게 ×0.95, 7일)를 더 구체적으로 사용. G03 중복
- 결정거리: G05 제거하고 G03만 쓸지, 또는 그 반대

### G04 — 주간 부하 +10% 체크
- 무엇: 이번주 / 지난주 > 10% → 부상 위험 경고
- 왜 안 부름: 추천 1회 호출이라 주간 비교 못 함
- 어디서 호출돼야: 주말 자동 분석 진입

### G06 — 고피로 시 강도까지 동시 감축
- 무엇: RPE 평균 > 9 + 수면 < 6h + 관절통 → "고피로 모드" → 세트 ½ + 무게 ½
- 왜 안 부름: 디로드 모드 분기가 진입에 없음. 항상 G05(일반 모드)만
- 결정거리: 사용자 컨디션 입력으로 디로드 모드 동적 분기할지

### G08 — MRV 도달 시 강제 디로드
- 무엇: 주간 세트 ≥ MRV + RPE drift ≥ 1 → 강제 디로드. 또는 6주 초과 시 강제
- 왜 안 부름: 누적 데이터 추적이 진입에 없음
- 어디서 호출돼야: 매 주말 자동 분석 진입

### G09 — 활성 디로드 vs 완전 휴식
- 무엇: 부상·번아웃·수면<5h 지속 → 패시브 1주 / 그 외 → 활성 디로드
- 왜 안 부름: 사용자에게 디로드 모드 묻지 않음
- 결정거리: 디로드 직전에 사용자에게 모드 선택 묻기 vs 자동 분기

### G11 — 미달 누적 디로드 트리거
- 무엇: 다관절 메인리프트 ≥ 2개 stall + 평균 RPE 9~10 → 디로드
- 왜 안 부름: stall 추적이 진입에 없음
- 어디서: 매 세션 후 분석

## 부상 복귀 후속 흐름 (Q02·Q03)

### Q02 — 부상 복귀 단계 전환
- 무엇: 5단계(휴식 → ROM → 무통 저강도 → 점진 부하 → 복귀). 통증 0/10 확인 후 다음 단계
- 왜 안 부름: 진입은 Q04/Q05/Q06으로 단계만 결정하고, 단계 간 전환 로직(Q02)은 안 부름
- 어디서 호출돼야: 부상 사용자의 매 세션 후 "다음 단계 진행 가능?" 체크 진입

### Q03 — 운동 복귀 기준 (4가지 충족)
- 무엇: ROM 정상 + 좌우 근력차 < 10% + 무통 컴파운드 가능 + 2주 무통 → 복귀 허용
- 왜 안 부름: 복귀 판정 자체를 추천이 안 함
- 결정거리: 부상 사용자가 "복귀 준비됐어요" 입력 시 Q03 4가지 체크리스트 띄울지

## 워밍업 다양화 (I07·I08·P02·E02·E09)

### I07 — 아이솔/머신 워밍업
- 무엇: 50% × 8렙 + 80% × 3렙 (1~2 워크업 세트면 충분)
- 왜 안 부름: 진입은 I06(컴파운드 워크업)만 사용. 아이솔 운동에 워크업 처방 없음
- 결정거리: 모든 운동에 워크업 줄지, 컴파운드만 줄지

### I08 — 시간 부족 시 무거운 워크업
- 무엇: 80% × 4렙 1세트만 → 본 세트
- 왜 안 부름: 시간 부족 분기가 진입에 없음
- 결정거리: 세션 가용시간이 짧으면 I08로 자동 전환할지

### P02 — PNF 스트레칭
- 무엇: Contract-relax-contract 기법, ROM 향상에 정/동적보다 우월
- 왜 안 부름: 진입은 정적/동적/SMR/모빌리티만 안내. PNF는 파트너 필요해서 디폴트에서 빠짐
- 결정거리: PNF를 옵션으로 노출할지

### E02 — Load-Velocity 워밍업 캘리브레이션
- 무엇: 워밍업 무게×속도 측정 → 그날 1RM 추정 → 본 세트 무게 ±5% 자동 보정
- 왜 안 부름: VBT 센서 필요
- 결정거리: VBT 센서 사용자 옵션으로

### E09 — RAMP 워밍업
- 무엇: R(심박↑)-A(타깃 활성)-M(가동성)-P(점증세트) 4단계 더 정밀한 워밍업
- 왜 안 부름: I05(일반)+I06(컴파운드 워크업)+P03(SMR)+P04(모빌리티)로 사실상 RAMP를 분리해서 이미 처방. RAMP는 통합 진입이 필요한데 진입 함수가 안 묶음
- 결정거리: I05+I06+P03+P04 통합해서 RAMP로 묶을지, 분리 유지할지

## RPE 표 처방 (rpe_meta.py: A02·A05·A07·A10·A13)

> **큰 결정거리**: **추천에서 RPE 표 기반 구체 무게 처방을 하나도 안 함**. 시작 무게는 D11/D12(빈 바벨) 또는 1RM × 0.8만. 사용자가 "5렙 RPE 8 목표면 80kg" 같은 구체 처방을 못 받음

### A02 — RPE × 렙 → 무게 직접 처방
- 무엇: 1RM × A07(렙·RPE) / 100
- 왜 안 부름: 추천은 렙범위·%1RM 범위만 출력 (예: "6~12렙, 67~85%"). 구체 무게는 사용자에게 떠넘김
- 결정거리: 종목별·세트별 구체 무게까지 처방할지

### A05 — 자율조절 vs 고정 % 모드 선택
- 무엇: 스트렝스 → 자율조절(RPE 보고 매 세션 무게 조정) / 근비대 → 볼륨 우선 (고정 %)
- 왜 안 부름: 진입에서 모드 선택 안 함
- 결정거리: 사용자에게 모드 노출할지, 자동 결정할지

### A07 — RTS 매트릭스 (렙×RPE → %1RM 표)
- 무엇: Tuchscherer 표 (1렙 RPE10=100% / 5렙 RPE8=81% / 8렙 RPE9=76% 등)
- 왜 안 부름: A02 도입 시 같이 살아남

### A10 — 종목·근육군별 변환 보정
- 무엇: 표 결과 % × 종목 보정 계수 (스쿼트 1.00 / 데드 1.01 / 단관절 0.95). 8렙 초과·단관절은 -3%p 추가
- 왜 안 부름: A02·A07 안 부르면 보정할 대상이 없음
- A02 도입 시 같이 활성화

### A13 — 세션 간 RPE drift 보정
- 무엇: 같은 무게×렙에서 세션마다 RPE +1 → -5% / -1 → +2.5~5%
- 왜 안 부름: 누적 데이터 + 세션 시작 시 호출 필요
- 어디서: 매 세션 시작 처방 진입
