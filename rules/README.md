## 룰 카드 한 장의 모양

```yaml
ID: A11
분류: RPE 자율조절
이름: 세트 간 무게 보정 (Helms 룰)
한줄설명: 목표 RPE 대비 0.5점 차이당 무게 ±2% (1점=±4%, 2점=±8%)
언제_발동: 세트 직후
필요한_정보: [목표 RPE, 입력 RPE, 현재 무게]
하는_일: 다음 세트 추천 무게 (kg)
근거:
  출처: Helms (Muscle and Strength Pyramid) / Ripped Body
  주소: https://rippedbody.com/rpe/
  신뢰도: ★★★
적용_위치:
  매트릭스:
    - {목표: 벌크업, 모듈: 세트 후 학습}
    - {목표: 스트렝스, 모듈: 세트 후 학습}
    - {목표: 근육량 증가, 모듈: 세트 후 학습}
  별도_레이어: []
```

각 필드의 의미는 [`../DSL.md`](../DSL.md) §2에서 한 줄씩 설명합니다.
